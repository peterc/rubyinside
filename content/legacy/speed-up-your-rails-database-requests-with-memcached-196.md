<p>Geoffrey Grosenbach looks at <a href="http://nubyonrails.topfunky.com/articles/2006/08/17/memcached-basics-for-rails">how to use memcached</a>, a fast in-memory caching daemon, from Ruby and Rails to speed up common repeated data operations, including ActiveRecord lookups. He also includes a useful install script and patch for Mac OS X users to make memcached fly on that platform. He also demonstrates the use of the <em>cached_model</em> gem to significantly speed up database reads from Rails applications.</p>
