There's been talk about producing something like this in the Rails community, and it seems someone has finally done it.. a <a href="http://rav.rubyforge.org/">Rails Application Visualizer.</a> Simply run a rake task and it produces a graphic showing your models and how they relate to one another. It requires the GraphViz library to run, and is only in its early days. Looks promising though!</p>
