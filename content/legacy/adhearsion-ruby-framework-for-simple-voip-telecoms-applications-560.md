<p><a href="http://adhearsion.com/">Adhearsion</a> is an open-source, telecommunications application framework that uses Ruby, coupled with <a href="http://asterisk.org/">Asterisk</a>, the popular open source telephony platform. It lets you build telecommunications systems and scripts in a quick and powerful way:</p>
<p><img src="/assets/2007/07/adhearsion2.jpg" height="182" width="294" border="1" hspace="4" vspace="4" alt="Adhearsion2"></p>
<p>Creator Jay Philips has written a chapter for O'Reilly forthcoming <a href="http://www.oreilly.com/catalog/9780596510480/index.html">"Asterisk: The Future of Telephony"</a> about Adhearsion, and O'Reilly have published <a href="http://www.oreillynet.com/pub/a/etel/2007/06/19/sophisticated-asterisk-development-with-adhearsion.html">an extended excerpt of it online</a>. It covers the installation of Adhearsion, basic script writing, database integration, and how to put together "micromenus" that operate in a vendor-independent fashion.</p>
<p>Learn more about the framework at the <a href="http://adhearsion.com/">official Adhearsion site</a>, and enjoy twisting together your phone system with a little Ruby!</p>
