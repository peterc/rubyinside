<p><img src="/assets/2010/12/apple-ruby-3.jpg" width="69" height="83" alt="apple-ruby-3.jpg" style="float:left; margin-right:12px; margin-bottom:12px;">MacRuby's lead developer and Apple employee Laurent Sansonetti has today <a href="http://www.macruby.org/blog/2011/03/23/macruby010.html">released MacRuby 0.10</a> (yep, that's <em>ten</em>), the latest version of the Mac OS X-focused Ruby implementation. 0.10 is the latest stepping stone on the way to a forthcoming 1.0 release.</p>
<h3>Getting MacRuby</h3>
<p>You can grab MacRuby 0.10 from <a href="http://www.macruby.org/downloads.html">the downloads page</a> or directly at <a href="http://www.macruby.org/files/MacRuby%200.10.zip">http://www.macruby.org/files/MacRuby 0.10.zip</a>. Beware, though, that the binary installer download will only work on 64 bit Intel-powered machines running OS X 10.6 or higher.</p>
<h3>New Features, New Possibilities</h3>
<p>0.10 is not a <em>major</em> release but a few things stand out in the <a href="http://lists.macosforge.org/pipermail/macruby-devel/2011-March/007390.html">release notes for 0.10</a> amongst all the usual performance tweaks and bug fixes:</p>
<ul>
<li>Support for the new MacBook Pro hardware (SandyBridge processors).</li>
<li>Fixes in macruby_deploy for App Store submissions.</li>
<li>Xcode4 support.</li>
</ul>
<p>This is the first time I've seen an obvious reference to the App Store in the MacRuby release notes and it's an exciting development. Back in October 2010, I wrote <a href="macruby-and-the-mac-app-store-3922.html">MacRuby + Mac App Store == Low Hanging Fruit for Rubyists</a> where I riffed on the possibilities that MacRuby could offer to Rubyists looking to make a splash with Ruby powered desktop apps. It looks like the door might be opening a little on this.</p>
<h3>Going Further</h3>
<p>If MacRuby interests you and you want to 'book up' check out <a href="http://oreilly.com/catalog/0636920000723/"><b>MacRuby: The Definitive Guide</b></a> by Matt Aimonetti and <b><a href="http://www.manning.com/lim/">MacRuby in Action</a></b> by Brendan Lim. Both are still in pre-release stages but beta copies are available.</p>
<p>I also recommend reading <a href="http://redwoodapp.posterous.com/macruby-for-the-desktop-six-reasons">MacRuby for the Desktop: Seven Reasons</a> by Andre Lewis of Scout. He's in the process of building a desktop Mac app with MacRuby and shares some general insights into how he's finding MacRuby.</p>
