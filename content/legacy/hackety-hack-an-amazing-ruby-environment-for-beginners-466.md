<p><img src="/assets/2007/04/hacketyhack.jpg" height="112" width="230" border="1" hspace="4" vspace="4" alt="Hacketyhack"></p>
<p><a href="http://whytheluckystiff.net/">Why The Lucky Stiff</a> is set to guarantee himself a place in the Programming Hall of Fame with his new project, <a href="http://hacketyhack.net/">Hackety Hack</a>, an uber-simple programming environment for kids, beginners, and amateur coders. It attempts to resolve a problem brought up in an old article of why's, <a href="http://whytheluckystiff.net/articles/theLittleCodersPredicament.html">"The Little Coder's Predicament"</a>, which lamented the ability for kids to quickly get coding on modern platforms, unlike in the Commodore 64 days.<br>
<img src="/assets/2007/04/hacketyhack2.jpg" height="306" width="356" border="1" hspace="4" vspace="4" alt="Hacketyhack2"></p>
<p>Currently the system is in beta and only available for Windows, but OS X and Linux are to follow. Why's ingenuity even goes as far as the installation program which makes it really easy for users to specify where they want their programs to be stored. </p>
<p><img src="/assets/2007/04/hacketyhack3.jpg" height="355" width="338" border="1" hspace="4" vspace="4" alt="Hacketyhack3"></p>
<p>As with any of Why's works, words cannot do it justice, so head on over to the <a href="http://hacketyhack.net/">official Hackety Hack site</a> and check it all out.</p>
