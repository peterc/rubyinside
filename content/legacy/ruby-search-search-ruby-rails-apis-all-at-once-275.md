<a href="http://labs.parkerfox.co.uk/ruby.search/">Ruby Search</a>, a project by Simon Parker, is a special Web search tool that looks through the Rails class index, Rails methods, Ruby standard libraries, and <em>Programming Ruby</em> and presents the results in a simple sidebar to be viewed in a frame on the right. It seems to be a bit patchy from my tests, but it's still a cool tool. <em>(Found via </em><em><a href="http://www.ozmm.org/">ozmm</a></em><em>)</em></p>
