<p><img alt="Aptana IDE features" src="/assets/2007/05/aptana1.jpg"></p>
<p><a href="http://www.aptana.org/">Aptana IDE</a> has been since inception targeted to Ajax developers and power web designers, with advanced features such as JavaScript debugging and built-in searchable reference manuals for many web standards (in addition to JavaScript, DOM, CSS, and HTML, it also bundles documentation for several Ajax libraries including <a href="http://script.aculo.us/">Scriptaculous</a>.)</p>
<p>With the merge of <a href="http://www.aptana.com/docs/index.php/RadRails">RadRails</a> into Aptana, the <a href="http://www.aptana.com/docs/index.php/Integrating_RadRails_with_Aptana">Aptana+RadRails bundled IDE</a> now supports Ruby syntax highlighting, debugging, code completion, and support for Rails-specific tasks like defining WEBrick and <a href="http://mongrel.rubyforge.org/">Mongrel</a> servers. Although the project is still in beta, it is already very usable for daily use.</p>
<p><em>Alternatives:</em> <a href="http://www.easyeclipse.org/">EasyEclipse</a> for Ruby on Rails, which is very easy to install (bundles the Java runtime) has RadRails built-in and you can <a href="http://www.aptana.com/docs/index.php/Plugging_Aptana_into_an_existing_Eclipse_configuration">enhance it using the Aptana plug-in</a>. Sun's <a href="http://www.netbeans.org/">NetBeans</a> 6 IDE bundles Ruby support (currently pre-release, but usable.) <a href="http://www.scintilla.org">SciTE</a> has built-in Ruby support and loads very quickly.</p>
