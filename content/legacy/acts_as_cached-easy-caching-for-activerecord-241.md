<a href="http://errtheblog.com/post/27">acts_as_cached</a> is a plugin by Chris Wanstrath, Tim Myrtle, and PJ Hyett that simply allows you to cache any Ruby object in memory (using <a href="http://www.danga.com/memcached/">memcached</a>). Check out <a href="http://require.errtheblog.com/filedetails.php?repname=plugins&amp;path=%2Facts_as_cached%2FREADME">the documentation for the full instructions</a>, but if you've got a working memcached server ready and waiting, it's as easy as installing a gem (<em>memcached-client</em>), a plugin (<em>acts_as_cached</em>), and adding '<em>acts_as_cached</em>' to your model.</p>
<p>Chris gives a full run down of <em>acts_as_cached</em> in <a href="http://errtheblog.com/post/27">this blog post</a>, and also links to a <a href="http://errtheblog.com/static/pdfs/memcached.pdf">great PDF presentation</a> about the plugin.</p>
