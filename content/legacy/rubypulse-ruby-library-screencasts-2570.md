<p><img src="/assets/2009/10/rubypulse.png" width="80" height="82" alt="rubypulse.png" style="float:left; margin-right:12px; margin-bottom:12px;"><a href="http://www.rubypulse.com/">RubyPulse</a> is a screencast site that publishes regular videos about different Ruby libraries. It's a month old so far and German Ruby developer <a href="http://github.com/aaalex">Alex Peuchert</a> has put out 9 episodes already. With a little support, perhaps Alex will be able to keep it going and become the pure Ruby equivalent of <a href="http://railscasts.com/">Ryan Bates!</a></p>
<p>Each RubyPulse episode is in Quicktime format and about 4 minutes long - each is also available in an iPod/iPhone tailored format. The nine episodes of RubyPulse so far include:</p>
<ul>
<li>
<a href="http://www.rubypulse.com/episode-0.1-fastri.html"><b>FastRI</b></a> - A look at the performance-focused alternative to the "ri" command line tool.</li>
<li>
<a href="http://www.rubypulse.com/episode-0.2-memoize.html"><b>Memoize</b></a> - An overview of memoization in Ruby (the process of caching method responses for future use).</li>
<li>
<a href="http://www.rubypulse.com/episode-0.3-gibbler.html"><b>Gibbler</b></a> - An exploration of the Gibbler library, <a href="gibbler-git-like-hashes-and-history-for-ruby-objects-1980.html">previously covered</a> on Ruby Inside.</li>
<li>
<a href="http://www.rubypulse.com/episode-0.4-sketches.html"><b>Sketches</b></a> - Sketches is a Ruby tool that lets you create and edit Ruby code in your favorite text editor while having it safely reloaded in IRB whenever the code is saved/updated.</li>
<li>
<a href="http://www.rubypulse.com/episode-0.5-rvm.html"><b>Ruby Version Manager</b></a> - A look at the "makes it easy to run multiple Ruby implementations at once" tool, <a href="rvm-ruby-version-manager-2347.html">RVM</a>.</li>
<li>
<a href="http://www.rubypulse.com/episode-0.6-hijack.html"><b>hijack</b></a> - A look at <a href="hijack-get-a-live-irb-prompt-for-any-existing-ruby-process-2232.html">hijack</a>, a tool that provides an <a href="irb-lets-bone-up-on-the-interactive-ruby-shell-1771.html">IRB</a> session into an existing Ruby process.</li>
<li>
<a href="http://www.rubypulse.com/episode-0.7-random_data.html"><b>random_data</b></a> - A library that provides random test data, including: names, addresses, dates, phone numbers, e-mail addresses, etc.</li>
<li>
<a href="http://www.rubypulse.com/episode-0.8_wirble_and_colored.html"><b>wirble and colored</b></a> - A look at a set of <a href="wirble-tab-completion-and-syntax-coloring-for-irb-336.html">IRB enhancements</a>. <b>(Bonus: Check out</b> <a href="irb-lets-bone-up-on-the-interactive-ruby-shell-1771.html"><b>our previous roundup of IRB stuff.</b></a><b>)</b>
</li>
<li>
<a href="http://www.rubypulse.com/episode-0.9_live_console.html"><b>live_console</b></a> - A look at a library that provides an IRB sesson to a process over a TCP or Unix socket connection. Unlike hijack, however, this has to be pre-arranged.</li>
</ul>
<p>Alex has done a great job with these videos so far and while his strong German accent <i>might</i> put some off, he's very effective at demonstrating how to use the libraries he covers quickly.</p>
