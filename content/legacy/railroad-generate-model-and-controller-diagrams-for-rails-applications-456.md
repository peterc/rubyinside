<p><a href="http://railroad.rubyforge.org/">RailRoad</a> is a Ruby application that generates model or controller diagrams for Ruby on Rails applications. It analyzes properties, inheritance, model associations, and so forth, and produces a graph description in the DOT language which can then be handled with third-party tools. The official site features <a href="http://railroad.rubyforge.org/#examples">several compelling examples</a>, including model and controller maps of the Typo blogging system (as seen above).</p>
