<p>Just over two months ago I <a href="hobo-a-rapid-web-app-builder-for-rails-350.html">posted about</a> a new rapid Web application development framework for Rails called <a href="http://www.hobocentral.net/">Hobo</a>. The team have now let me know they've got a brand new, redesigned site, and that Hobo has a major, new release.</p>
<p>Hobo makes it really easy to get a basic Web application up and running in Rails. It's a little like scaffolding but with a turbocharger under the hood. And if you want to get up and running in just a few minutes, they have a <a href="http://www.hobocentral.net/blog/getting-started">short tutorial</a> along with <a href="http://www.hobocentral.net/screencasts.php">three screencasts</a>.</p>
