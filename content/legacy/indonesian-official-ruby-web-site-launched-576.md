<p><em>(English translation of the text below follows)</em></p>
<p><a href="http://www.ruby-lang.org/id/">Situs Resmi Ruby Indonesia</a> telah <a href="http://ariekusumaatmaja.wordpress.com/2007/08/04/wwwruby-langorgid-rilis/">diluncurkan</a>. Jadi sekarang telah tersedia 9 terjemahan dalam situs web Ruby, belum termasuk beberapa lagi yang masih dalam pengerjaan.</p>
<p>Programmer Indonesia yang menyukai Ruby semakin banyak, berdiskusi menggunakan wadah <a href="http://groups.yahoo.com/groups/id-ruby">milis id-ruby</a>, yang sekarang sudah semakin ramai.</p>
<p>Adanya situs ini diharapkan dapat semakin membantu kemajuan Ruby dan teknologi di tanah air tercinta. Selain berisi materi yang terdapat di situs Ruby versi Inggris, situs ini juga menyediakan materi-materi yang spesifik terhadap komunitas Ruby di Indonesia.</p>
<p>Tim pelaksana proyek ini terdiri dari <a href="http://ariekusumaatmaja.wordpress.com/">Arie Kusuma Atmaja</a>, <a href="http://microjet.ath.cx/">Yohanes Santoso</a>, dan <a href="http://www.hendyirawan.com/">Hendy Irawan</a>.</p>
<p><em>Translation:</em></p>
<p><a href="http://www.ruby-lang.org/id/">Official Ruby web site in Bahasa Indonesia</a> has been <a href="http://ariekusumaatmaja.wordpress.com/2007/08/04/wwwruby-langorgid-rilis/">launched</a>.<br>
This brings the total number of translations available for the entire Ruby web site to 9 languages, in addition to several more translation projects that are still in progress.</p>
<p>The number Indonesian developers interested in Ruby has been growing, discussing topics through the <a href="http://groups.yahoo.com/groups/id-ruby">id-ruby mailing list</a> which is now more active than ever.</p>
<p>Native Indonesian translation of the Ruby official site is hoped to be beneficial to both Ruby and development of technology in Indonesia. In addition to standard content, the Indonesian site also provides Indonesian-specific materials and resources.</p>
<p>The team for this translation project consists of <a href="http://ariekusumaatmaja.wordpress.com/">Arie Kusuma Atmaja</a>, <a href="http://microjet.ath.cx/">Yohanes Santoso</a>, and <a href="http://www.hendyirawan.com/">Hendy Irawan</a>.</p>
