<p>The <a href="http://railstutorial.org/">Ruby on Rails Tutorial: Learn Rails by Example</a> (a.k.a. <em>railstutorial.org</em>) by <a href="http://michaelhartl.com/">Michael Hartl</a> has become a must read for developers learning how to build Rails apps. Michael has put together a great Rails 2.3 tutorial, releasing it all for free online chapter by chapter. Now, Michael's going three steps further:</p>
<p><strong>1 — A new, Rails 3.0 focused version.</strong> The free online book previously covered Rails 2.3 but Michael's updated it to cover Rails 3.0 too. He's also selling it as a DRM-free PDF for $39 (you get a PDF of the Rails 2.3 version too). As a gesture of goodwill to Ruby Inside's readers, he's made a coupon code that works till the end of August - it's <code><a href="http://railstutorial.org/?discount=rubyinside01">rubyinside01</a></code> and gets you 20% off (so a total of $31.20 in the end).</p>
<p><strong>2 — Creative Commons licensing of the existing online text.</strong> Like all of us, Michael needs to make some money, but a side benefit is that he's making the existing Rails 2.3 focused text Creative Commons licensed! This will allow you to distribute it, translate it, put snippets on your blog, and so forth. <em>(Update: Michael notes that this is sort of in the air at the moment pending more resources. He has more preparation to do to make this work properly, but the spirit is there.)</em></p>
<p><strong>3 — A print book, published by Addison-Wesley.</strong> A print edition, <a href="http://www.amazon.com/Ruby-Rails-Tutorial-Addison-Wesley-Professional/dp/0321743121"><em>Ruby on Rails 3 Tutorial: Learn Rails by Example</em></a>, is due out in the fall as part of the <a href="http://www.informit.com/promotions/promotion.aspx?promo=135393">Professional Ruby Series</a> (the same series as <a href="http://www.amazon.com/Rails-Way-Addison-Wesley-Professional-Ruby/dp/0321601661"><em>The Rails 3 Way</em></a> by Obie Fernandez), and is currently available for <a href="http://www.amazon.com/Ruby-Rails-Tutorial-Addison-Wesley-Professional/dp/0321743121">pre-order at Amazon</a>.</p>
<p>As a bit of a "geek aside", the <a href="http://railstutorial.org/"><em>Ruby on Rails Tutorial</em> book</a> is written using PolyTeXnic, a pure-Ruby markup system built on top of the <a href="http://www.latex-project.org/">LaTeX</a> typesetting system. PolyTeXnic converts a select subset of LaTeX to HTML, while also producing PDFs via the <tt>pdflatex</tt> command.</p>
<p>In addition to supporting code-heavy programming books such as <em>Ruby on Rails Tutorial</em>, PolyTeXnic can also produce mathematical documents; see, for example, Michael Hartl's anti-pi propaganda piece called <a href="http://tauday.com/"><em>The Tau Manifesto</em></a>. Michael hopes to release PolyTeXnic as an open-source project some time later this year. I keep nagging him about it.</p>
