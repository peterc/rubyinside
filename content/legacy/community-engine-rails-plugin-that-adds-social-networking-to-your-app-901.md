<p>Prolific Rails developer Bruno Bornsztein (<a href="interview-with-rails-development-addict-bruno-bornsztein-393.html">interviewed on Ruby Inside</a> in February 2007) does it again! Not content to settle for releasing 14-plus Rails related projects and Web sites, he has developed <a href="http://missingmethod.brunobornsztein.com/projects/community_engine/">Community Engine</a>, an "instant open-source social network plugin" for Rails. Unlike <a href="insoshi-open-source-rails-based-social-networking-platform-878.html">Insoshi</a> and <a href="http://lovdbyless.com/">Lovd By Less</a>, which are full social networking Rails applications, Community Engine is a plugin that can add social networking features to <span style="font-style: italic;">existing</span> Rails applications.</p>
<p>Community Engine provides authentication, profile, search, blogging (with tagging, categories and rich-text editing), photo, bookmarking, forum, and other similar functionality. Community Engine has been extracted from the features of two successful social networking sites, <a href="http://www.curbly.com/">Curbly</a> and <a href="http://www.uncooped.com/">Uncooped</a>, and has effectively been stress-tested by the general public for the last year.</p>
