<blockquote class="stylized"><p><a href="http://github.com/colszowka/simplecov">SimpleCov</a> is a code coverage analysis tool for Ruby 1.9. It uses 1.9’s built-in Coverage library to gather code coverage data, but makes processing it’s results much easier by providing a clean API to filter, group, merge, format and display those results, thus giving you a complete code coverage suite with just a couple lines of code.</p></blockquote>
<p><cite>Christoph Olszowka</cite></p>
<p><a href="http://github.com/colszowka/simplecov">SimpleCov</a> builds on the ideas presented in Aaron Patterson's recent, and awesome, <a href="http://engineering.attinteractive.com/2010/08/code-coverage-in-ruby-1-9/">Writing a Code Coverage Tool with Ruby 1.9</a> blog post. The result is a powerful yet straightforward code coverage tool that puts out some well formatted stats.</p>
