<p>Courtenay of the <a href="http://blog.caboo.se">caboo.se</a> has put together a solid "<a href="http://blog.caboo.se/articles/2007/4/8/sample-rails-application-updated">sample Rails application</a>" that enables you to get up and running with a new Rails application quickly.</p>
<p>It includes several useful features out of the box:</p>
<ul>
<li>views implemented with Yahoo! YUI CSS grids</li>
<li>timezone support</li>
<li>basic authorization system with RESTful authentication</li>
<li>polymorphic image model</li>
<li>around_filters with scoping</li>
<li>exception notification</li>
</ul>
<p>You can learn more about the sample app in Courtenay's <a href="http://blog.caboo.se/articles/2007/4/3/caboose-sample-rails-application">first post about it</a>. Even if you don't find the template directly useful, it's worth looking at to see how a proficient Rails developer will begin to construct an application from scratch.</p>
