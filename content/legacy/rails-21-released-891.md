<p>Hours ago, David Heinemeier Hansson <a href="http://twitter.com/d2h/statuses/808936705">announced</a> informally on Twitter:</p>
<blockquote>
<p><a href="http://github.com/rails/rails/commits/v2.1.0_RC1">Rails 2.1 RC1 has been tagged</a>, the gems are on the beta server, official announcement shortly. But no need holding you back from trying it.</p>
</blockquote>
<p>New features include built-in <a href="http://ryandaigle.com/articles/2008/1/25/what-s-new-in-edge-rails-easier-timezones">timezone support</a>, <a href="http://ryandaigle.com/articles/2008/4/1/what-s-new-in-edge-rails-gem-dependencies">Gem dependencies</a>, <a href="http://ryandaigle.com/articles/2007/12/19/what-s-new-in-edge-rails-pluggable-controller-caching">better caching</a>, and more.</p>
<p>To get Rails 2.1 RC1 from the beta gems server, just use:</p>
<blockquote>
<p>sudo gem install rails --source http://gems.rubyonrails.com/</p>
</blockquote>
<p>If you prefer to go native, Ryan Bates of Railscasts has already produced <a href="http://railscasts.com/episodes/105">a screencast showing how to install Rails 2.1 RC1 using Git</a>.</p>
<p>To keep up with the community chatter about Rails 2.1, check out <a href="http://summize.com/search?q=rails+2.1">this search for "Rails 2.1"</a> on Twitter Summize. There's already a lot of activity.</p>
