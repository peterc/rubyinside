<p><a href="http://blog.evanweaver.com/articles/2006/06/02/allison">Allison</a> is a new RDoc template produced by <a href="http://blog.evanweaver.com/">Evan Weaver</a>. I'm not entirely sure about the colors (but these are easily tweaked with CSS), but it still looks very nice and clean, and for many situations is an obvious improvement over the default framed template RDoc provides. Evan provides <a href="http://blog.evanweaver.com/">full instructions for using Allison</a> for your documentation.</p>
