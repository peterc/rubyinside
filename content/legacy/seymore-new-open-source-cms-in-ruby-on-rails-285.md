<p><a href="http://www.slicedsoftware.com/software/seymore/">Seymore</a> is a new Rails-based content management system developed by Thomas Mango and released to the public in the last couple of days. Six screencasts are provided demonstrating how the system looks and operates. Version 0.1 is available now as a complete build in .tar.gz and .zip formats and a more up to date trunk copy is available via SVN from <a href="http://code.google.com/p/seymore/source">Seymore's Google Code repository</a>.</p>
