<p><img src="/assets/2007/09/netbeans6.jpg" height="134" width="299" border="1" hspace="4" vspace="4" alt="Netbeans6"></p>
<p>NetBeans is an open-source IDE written in Java (somewhat like Eclipse, but not the same), and version 6.0 is the first with a Ruby-specific edition that focuses solidly on the things that Ruby and Ruby on Rails developers need. The NetBeans team have just released the first beta version.</p>
<p>NetBeans is a powerful and free. You can create Ruby and Rails projects, run Ruby files, configure interpreters (MRI and JRuby), install Gems graphically, run tests, run RSpecs, debug Ruby code, run Rails apps, and so on, all from the IDE. The Ruby edition is <a href="http://bits.netbeans.org/download/6_0/beta1/latest/">only a 19MB download</a> and it's available right now. There are several Ruby related <a href="http://www.netbeans.org/kb/60/flash.html">NetBeans screencasts</a> for the less convinced.</p>
<p>Also of interest will be George Cook's <a href="http://lifeonrails.org/2007/8/30/netbeans-the-best-ruby-on-rails-ide">summary of why NetBeans is "THE best Ruby on Rails IDE"</a> which features tons of screenshots and example uses.</p>
