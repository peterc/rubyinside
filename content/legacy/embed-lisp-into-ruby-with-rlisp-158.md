<p><a href="http://t-a-w.blogspot.com/">Tomasz Węgrzanowski</a> (aka <em>Taw</em>) has developed <a href="http://t-a-w.blogspot.com/2006/07/rlisp-lisp-naturally-embedded-in-ruby.html">a Lisp interpreter embedded in Ruby called RLisp</a>. It's early days and ultra-alpha-quality, but it's a cool project. You can find the <a href="http://zabor.org/taw/rlisp/">source code here</a>, and Taw is offering <a href="http://t-a-w.blogspot.com/2006/07/free-beer-for-first-10-cool-rlisp.html">free beer for the first 10 developers to write a cool RLisp program</a>. RLisp already has some basic OO support (and even <a href="http://t-a-w.blogspot.com/2006/07/rlisp-gets-http-support.html">HTTP support</a>) so it's fun to play with.</p>
