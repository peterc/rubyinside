<p>Richard White quotes an anonymous user of his <a href="http://www.ajaxscaffold.com/">AJAX scaffolding system</a> for Rails:</p>
<blockquote><p><em>I really like Ajax Scaffold but it’s really hard to upgrade when a new version comes out. I usually spend a couple hours diffing all my scaffolds with a newly generated one.</em></p></blockquote>
<p>Richard listened, and no longer are you forced to use AJAX Scaffold as a code generator.. it'll now work as a regular Rails plugin! Richard <a href="http://www.height1percent.com/articles/2006/09/12/ajax-scaffold-plugin-released">has more information here</a>. Nice work, Richard!</p>
