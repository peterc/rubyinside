<p>The <a href="http://jruby.codehaus.org/">JRuby</a> team has just announced the release of JRuby 0.9.2 (<a href="http://dist.codehaus.org/jruby/">download</a>). Despite being a minor point release, a lot of things have been tweaked and added. Direct from the team:</p>
<blockquote>
<p>This release has some great improvements:</p>
<p>Extensions openssl and readline now working<br>
Code for a new graphical irb console<br>
Partial support for iconv and bigdecimal extensions<br>
RSpec now supported<br>
Improved Rails support<br>
Fixed all known block and scoping bugs<br>
Enhanced parser performance<br>
More compiler and performance work<br>
Refactored variable scoping logic<br>
127 Jira issues resolved since 0.9.1</p>
</blockquote>
<p>Congratulations to the JRuby team for their continuing efforts!</p>
