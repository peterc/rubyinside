<p>Mike Potter of Adobe has posted <a href="http://blogs.adobe.com/mikepotter/2006/09/announcing_the_1.html">an announcement of the release of a Ruby on Rails SDK</a> for developing 'Rich Internet Applications' using Rails and Adobe's technologies. The code has <a href="http://code.google.com/p/rubyonrails-ria-sdk-by-adobe/">been made available</a> on Google's Developer Network.</p>
<p>The SDK includes four samples: a 'directory explorer', an 'issue tracker', a file transfer demo, and a 'send data' demo (appears to be a way to pipe file data from Rails into a Flex app). Derek Wischusen of the <a href="http://flexonrails.net/">Flex on Rails</a> blog has <a href="http://flexonrails.net/?p=12">some more info</a>.</p>
