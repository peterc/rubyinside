<p><a href="http://www.erikveen.dds.nl/tar2rubyscript/index.html">Tar2RubyScript</a> is a tool that packages up your entire Rails application into a single <em>tar</em> file with a special loader so that users can get up and running with one click.</p>
<p><small>The next step is working out how to package up an entire Rails app, database, and WEBrick (or Mongrel) daemon into a single OS X .app folder so that you can distribute cool Web apps to run on people's local machines. <a href="http://www.bigbold.com/snippets/">Snippets</a> would be a great candidate for this. I'll post more information when I figure it out!</small></p>
