<p><a href="http://mosquito.the.tool.googlepages.com/">Mosquito</a> is a new, automatable, testing tool that uses RubyCocoa and OS X to perform Web operations and take screenshots of each point along the way. It's a little like the Windows-only <a href="http://wtr.rubyforge.org/">Watir</a>. I can see the technology used by Mosquito also being useful for automating Web operations even without the testing focus.</p>
