<p><img src="/assets/2010/02/rbp.png" width="125" height="111" alt="rbp.png" style="float:left; margin-right:12px; margin-bottom:12px;"><a href="http://rubybestpractices.com/">Ruby Best Practices</a> is a book by Gregory Brown (and published by O'Reilly) that looks into the "Ruby way" of doing things in the Ruby language and, specifically, why Rubyists tend to write Ruby the way they do. It's an engaging book and <a href="ruby-best-practices-gregory-brown-interview-1332.html">we took a look at it and interviewed Gregory Brown</a> about it just over a year ago.</p>
<p>Gregory always wanted to be able to give away his book with a Creative Commons license eventually and O'Reilly have kindly allowed him to start doing that, a chapter at a time. So far, the <a href="http://github.com/sandal/rbp-book/tree/gh-pages/pdfs/">first five chapters are available</a> to download at GitHub (in PDF format). Specifically:</p>
<ul>
<li>Chapter 1 - <a href="http://github.com/sandal/rbp-book/raw/gh-pages/pdfs/ch01.pdf">Driving Code Through Tests</a>
</li>
<li>Chapter 2 - <a href="http://github.com/sandal/rbp-book/raw/gh-pages/pdfs/ch02.pdf">Designing Beautiful APIs</a>
</li>
<li>Chapter 3 - <a href="http://github.com/sandal/rbp-book/raw/gh-pages/pdfs/ch03.pdf">Mastering the Dynamic Toolkit</a>
</li>
<li>Chapter 4 - <a href="http://github.com/sandal/rbp-book/raw/gh-pages/pdfs/ch04.pdf">Text Processing and File Management</a>
</li>
<li>Chapter 5 - <a href="http://github.com/sandal/rbp-book/raw/gh-pages/pdfs/ch05.pdf">Functional Programming Techniques</a>
</li>
</ul>
<p>However, it's not just a free for all! If you read these chapters, Gregory (the primary author of the book) implores you to comment and offer any advice or insights you might have <a href="http://blog.rubybestpractices.com/">on the relevant posts on the RBP blog</a>.. he also suggests that if you <i>really</i> like it, you should buy a copy of the book somehow, if only to convince O'Reilly that this publishing model is a "Good Thing." Gregory <a href="http://blog.rubybestpractices.com/posts/gregory/015-rbp-ch1.html">claims</a> that the book hasn't even made its advance back yet (how!?), which makes this gesture all the better.</p>
<p>To buy a copy, <a href="http://oreilly.com/catalog/9780596523008/index.html">head over to O'Reilly for e-book and print copies</a> or go to your usual bookselling site (Amazon.com are probably the cheapest as usual).</p>
<p>Kudos to Gregory and O'Reilly for this move.</p>
