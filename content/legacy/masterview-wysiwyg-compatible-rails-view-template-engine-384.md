<p>Why summize when the <a href="http://masterview.org/">official MasterView site</a> has the best explanation I can find?</p>
<blockquote><p>MasterView™ is a ruby/rails optimized HTML/XHTML friendly template engine. It is designed to use the full power and productivity of rails including layouts, partials, and rails html helpers while still being editable/styleable in a WYSIWYG HTML editor.</p></blockquote>
<p>MasterView allows you to write HTML in a somewhat normal way for an entire page (that is, one page within a single HTML file) along with specialized attributes which MasterView can then use to convert that HTML file into a set of Rails-friendly views and partials. The <a href="http://masterview.org/simple_diagram.html">fully zoomed version</a> of the screenshot on this post provides a good demonstration. The Masterview site also provides <a href="http://masterview.org/videos/demo_short.html">some</a> <a href="http://masterview.org/videos/masterview_quick_start.html">screencasts</a> to show how to get things going.</p>
<p>Yet another interesting Rails templating option!</p>
