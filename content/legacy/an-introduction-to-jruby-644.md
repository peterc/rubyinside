<p><a href="http://www.builderau.com.au/program/java/soa/JRuby-An-Introduction/0,339024620,339283504,00.htm">JRuby: An Introduction</a> is a fresh new article by Chris Duckett that presents a practical, code-driven introduction to JRuby, a Java implementation of Ruby. It's the perfect introduction for people who, perhaps, have an idea of what JRuby is, but who haven't yet tried out any of the Java connectivity. The first page looks at interacting with basic Java libraries / classes, with the second moving on to building a basic Swing-powered GUI app using Java APIs from Ruby code.</p>
<p><strong>Added: </strong>There's also <a href="http://podcast.sdruby.com/2007/11/2/episode-034-intro-to-jruby">a screencast by Brian Chapados called Intro to JRuby</a> that's worth watching for a more practical, real-time look.</p>
