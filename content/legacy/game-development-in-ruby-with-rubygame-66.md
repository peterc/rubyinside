<p><a href="http://rubygame.seul.org/">Rubygame</a> is a new library that provides ties between the <a href="http://www.libsdl.org/index.php">SDL</a> (Simple DirectMedia Layer) and Ruby, so that you can do graphics and sound work from Ruby, ultimately to build games. It's stylized off of the popular <a href="http://www.pygame.org/">pygame</a>, the equivalent library for Python. Unfortunately it currently only works on Linux (and other Unix-like systems) but Windows and Mac users should be in luck soon.</p>
<p>If playing with OpenGL, 3D graphics, and building some simple games sounds appealing to you, <a href="http://rubygame.seul.org/">Rubygame</a> is the place to be.</p>
