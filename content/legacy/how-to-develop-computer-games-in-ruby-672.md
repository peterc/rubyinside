<p>In the first of a series, Andrea Wright looks at <a href="http://www.oreillynet.com/pub/a/ruby/2007/12/04/creating-games-in-ruby.html">how to create games using Ruby</a>. The series isn't a tutorial level set of articles, but instead a walk through some of the major libraries and technologies you can use along with some code examples. In this first edition, Andrea looks at <a href="http://www.kmc.gr.jp/~ohai/rubysdl.en.html">Ruby/SDL</a>, <a href="http://rubygame.sourceforge.net/info.html">Rubygame</a> (inspired heavily by <a href="http://www.pygame.org/news.html">PyGame</a>, a significant Python game development library) and <a href="http://sourceforge.net/projects/rudl/">RUDL</a>.</p>
