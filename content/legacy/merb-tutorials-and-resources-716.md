<p><img src="/assets/2008/02/merb.png" alt="merb.png" height="67" width="168"></p>
<p><a href="http://merbivore.com/">Merb</a> is a Ruby Web framework that most Ruby and Rails developers are familiar with, even if they haven't used it. Merb is a Model View Controller based framework, somewhat like Rails, but it's significantly lighter, faster, and more customizable. Merb's strength is that it has a lightweight core and relies on plugins, selected by the user, to provide most of the extended functionality.</p>
<p>This post is to summarize some of the more interesting Merb tutorials and references, so that newcomers to Merb, or those who want to refresh their knowledge, can get started quickly:</p>
<p><big><strong>Official Stuff</strong></big></p>
<p style="margin-left: 2em"><a href="http://merbivore.com/">Merb's official homepage</a> - The typical launching point for learning about what Merb is, and for accessing stable and trunk builds or learning how to contribute to the project.</p>
<p><a href="http://merbivore.com/documentation.html">Merb's official API documentation</a> - RDoc generated documentation, straight from Merb's source code. The main page demonstrates how to install Merb, get a basic application started, and how the main concepts, such as models, views, and controllers, work.</p>
<p><a href="http://merbivore.com/features.html">Merb Features</a> - The official list of Merb's features.</p>
<p><a href="http://brainspl.at/articles/2008/01/10/merb-0-5-0-is-out">Merb 0.5 Release Notes</a> - The release notes for the latest (as of February 2008) release of Merb.</p>
<p><a href="http://github.com/wycats/merb-core/tree/master">Merb's source code repository trunk</a> - A view of Merb's official git repository at github.com. If you want the latest version of Merb, and live on the edge, get it from here.</p>
<p><big><strong>Motivation / Rationale / Advocacy</strong></big></p>
<p style="margin-left: 2em"><a href="http://railspikes.com/2007/4/1/merb">Merb (and why you potentially should care)</a> - A well written demonstration of why Merb matters and Merb's key features.</p>
<p><a href="http://rethink.unspace.ca/2007/12/3/merb-tastic">Merb-tastic</a> - A general overview of Merb from a non-official point of view. Unspace, a Canadian Ruby consulting firm, takes a look at what Merb is, how it can use different ORMs, and what their motivations for using Merb are.</p>
<p><big><strong>Introductions / Tutorials / Installation</strong></big></p>
<p style="margin-left: 2em"><a href="http://codingbitch.com/p/comboy/merb%20+%20datamapper%20+%20noob:%20quick%20start">Merb + DataMapper + Noob: Quick Start</a> - A great introductory tutorial for newcomers to Merb, demonstrating how to create a note taking / simple blog application from start to finish.</p>
<p><a href="http://www.luretanker.no/blog/2007/06/19/merb-tutorial/">Merb Tutorial</a> - Another tutorial that's older, and partly out of date, but the supplied source code has been updated to the latest Merb standards (well 0.4.2, anyway).</p>
<p><a href="http://rorblog.techcfl.com/2008/02/01/merborial-getting-started-with-merb-and-datamapper/">Merborial: Getting Started with Merb and DataMapper</a> - Chris Kaukis with the first installment of a very up to date Merb tutorial (February 2008). In this installment, he shows how to install Merb, DataMapper, and some other plugins, sets up an initial Merb application, creates a database, and provides a bunch of links to useful resources.</p>
<p><a href="http://jit.nuance9.com/2007/11/merb-datamapper-getting-rolling.html">Merb &amp; DataMapper: Getting Rolling</a> - A quick introductory piece to installing Merb and DataMapper by Justin Pease.</p>
<p><big><strong>Doing Stuff with Merb</strong></big></p>
<p style="margin-left: 2em"><a href="http://brainspl.at/articles/2007/12/02/a-quick-jaunt-through-merbs-framework-code">A Quick Jaunt Through Merb's Framework Code</a> - A blog post by Ezra Zygmuntowicz that takes a look at how Merb processes and handles incoming requests.</p>
<p><a href="http://www.oreillynet.com/pub/a/ruby/2008/01/14/shoes-meets-merb-interfacing-a-gtk2-front-end-and-a-rails-web-service.html">Shoes Meets Merb: Driving A GUI App through Web Services</a> - A six page, in-depth article by Gregory Brown and Brad Ediger that demonstrates how to use the Shoes GUI toolkit along with Merb, where Merb provides the data, and Shoes provides the interface.</p>
<p><a href="http://www.benr75.com/articles/2007/12/24/taking_merb_to_production_nginx_evented_mongrel_capistrano">Taking Merb to Production</a> - Ben Reubenstein looks at how to use Capistrano and Nginx to deploy Merb applications.</p>
<p><a href="http://hassox.blogspot.com/2008/01/merbful-authentication-is-born.html">Merbful Authentication</a> - A look at a new (as of January 2008) plugin for Merb that provides an easy to use authentication system.</p>
<p><a href="http://toolmantim.com/article/2008/1/22/merb_routing_in_0_5">Merb Routing in 0.5</a> - A great look at how the URL routing system works in Merb 0.5 by Toolman Tim!</p>
<p><a href="http://www.scribelabs.com/2007/10/18/file-uploads-via-merb-inside-a-rails-application-part-1-of-3/">File Uploads Via Merb Inside A Rails Application</a> - A three part tutorial showing how to use Merb alongside a Rails application to handle file uploads in an efficient manner. This is what Merb was originally created for.</p>
<p><a href="http://blog.vixiom.com/2007/06/29/merb-on-air-drag-and-drop-multiple-file-upload/">Merb on AIR - Drag and Drop Multiple File Upload</a> - A slightly old (June 2007) tutorial demonstrating how to use Merb alongside an Adobe AIR powered client to handle file uploads.</p>
<p><big><strong>The Future and More!</strong></big></p>
<p style="margin-left: 2em"><a href="http://www.infoq.com/news/2008/01/road-to-merb">The Road to Merb 1.0</a> - Robert Bazinet of InfoQ interviews Merb creator Ezra Zygmuntowicz about the future for Merb.</p>
<p><a href="http://yehudakatz.com/2008/01/14/merbnext/">Merb.next</a> - Yehuda Katz, a developer on the Merb team, looks at how things are progressing with Merb up to version 1.0.</p>
<p><a href="http://www.merbcast.com/tutorials.html">A Mega List of Even More Merb Tutorials</a> - For those readers who aren't satisfied enough by the above list of Merb links, Merbcast.com has put together even more for you!</p>
