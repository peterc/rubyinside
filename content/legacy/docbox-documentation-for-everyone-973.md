<p><a href="http://docbox.inspir.es">DocBox</a> is the latest attempt to improve the Ruby/Rails documentation scene.  Created as a Google Summer of Code project by <a href="http://inspir.es">Ian Ownbey</a> (mentored by <a href="http://www.omgbloglol.com/">Jeremy McAnally</a>) DocBox "sits on top of RDoc and allows users to update documentation through a wiki-like interface." Changes to the 'wiki' are then folded back into your code and committed to a git documentation branch.</p>
<p>While still young, this project shows a lot of promise, as it allows anyone to write detailed, versioned, documentation without having to download or view any source code.</p>
<p>Naturally, DocBox is open source and is available <a href="http://github.com/iownbey/docbox/tree/master">at GitHub.</a> Jeremy and Ian <a href="http://www.omgbloglol.com/?p=35">both</a><span style="#000000;"> </span><a href="http://inspir.es/posts/8-docbox">encourage</a> people to jump in.</p>
