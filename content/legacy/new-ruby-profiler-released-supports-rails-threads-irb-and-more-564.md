<p>Charlie Savage has <a href="http://cfis.savagexi.com/articles/2007/07/09/announcing-ruby-prof-0-5-0">released the 0.5.x family</a> of <a href="http://rubyforge.org/projects/ruby-prof">ruby-prof</a>, a profiling library for Ruby. We <a href="new-version-of-the-ruby-profiler-ruby-prof-available-110.html">last posted</a> about ruby-prof over a year ago, but 0.5 is a <em>major</em> leap forward with improved or new-found support for:</p>
<ul>
<li>recursive methods</li>
<li>Rails</li>
<li>threaded applications</li>
<li>calling from IRB</li>
<li>64 bit support</li>
</ul>
<p>Learn more in <a href="http://cfis.savagexi.com/articles/2007/07/09/announcing-ruby-prof-0-5-0">the release post</a>.</p>
