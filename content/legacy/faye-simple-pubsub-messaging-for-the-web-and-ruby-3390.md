<blockquote class="stylized"><p>Faye is an easy-to-use publish-subscribe messaging system based on the Bayeux protocol. It provides message servers for Node.js and Rack, and clients for use in Node and Ruby programs and in the browser.</p></blockquote>
<p><cite>James Coglan</cite></p>
<p>This is a simpler, open source equivalent to systems like New Bamboo's <a href="http://pusherapp.com/">Pusher</a>, but using the Comet-like <a href="http://svn.cometd.com/trunk/bayeux/bayeux.html">Bayeux</a> protocol with AJAX rather than WebSockets. Start a server (based on node.js or Ruby/Rack) and then push and pull messages between clients and apps in JavaScript and Ruby.</p>
