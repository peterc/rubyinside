<p>Dr Nic Williams has been busy playing with iPhone and Objective C development lately and, unsurprisingly, has found a way to bring Ruby into the mix. He has developed <a href="http://drnicwilliams.com/2008/07/04/unit-testing-iphone-apps-with-ruby-rbiphonetest/">rbiphonetest</a>, a Ruby-based testing framework for iPhone / Objective C applications that uses <a href="the-ultimate-list-of-rubycocoa-tutorials-tips-and-tools-728.html">RubyCocoa</a> to provide the necessary bridge. As well as producing an in-depth 20 minute demonstration <a href="http://www.vimeo.com/1262916?pg=embed&amp;sec=1262916">screencast</a>, he has also packed some key information into <a href="http://drnicwilliams.com/2008/07/04/unit-testing-iphone-apps-with-ruby-rbiphonetest/">his blog post</a>.</p>
<p>If you're a Rubyist with ambitions of developing native iPhone applications using Objective C (and perhaps even Ruby one day), it's certainly worth a look (as are all of Dr Nic's projects, of course).</p>
