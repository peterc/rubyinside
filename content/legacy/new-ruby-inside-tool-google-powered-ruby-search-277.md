<p>Hot on the heels of the <a href="ruby-search-search-ruby-rails-apis-all-at-once-275.html">last post</a> comes a new feature to Ruby Inside.. a Ruby and Rails search engine powered by Google. It's on <a href="index.html">the front page here at Ruby Inside</a> (link for those reading via RSS) and lets you search a bumper collection of Ruby and Rails sites in one go.<br>
If you search for 'Integer' you get the Ruby class references, rather than the Java and Wikipedia links you get with Google.com, and you can use it as a quick, reliable way to pin down the Ruby information you need. So far there are 23 sites in the search engine, including the main reference sites, the Rails forum, and several key weblogs, but I'm adding more sites over time. Leave a comment against this post if you want to suggest a site for the search engine.</p>
