<p>High off Baltimore Pandemic and Yellow Tops, I believe we promised a release candidate shortly after RailsConf. <p>Now it’s time to just say good is good enough, otherwise we could keep on with this forever. So please welcome the Rails 3 release candidate! You install, as always, with <code>gem install rails --pre</code></p>
