<p>The <a href="http://www.robotcoop.com/">Robot Co-op</a>, the team behind <a href="http://www.43things.com/">43things</a>, has released <a href="http://dev.robotcoop.com/Libraries/">several great Ruby libraries</a> for free, all available as gems. Here are some of the highlights:</p>
<blockquote>
<p><strong>cached_model:</strong> A subclass of Rails' ActiveRecord::Base that caches objects in memory using memcached. Big speed improvements in many circumstances.</p>
<p><strong>geocoder-us:</strong> A library that accesses geocoder.us's geocoding API that lets you look up addresses and turn them into coordinates.</p>
<p><strong>google-geocode:</strong> A library that uses the Google geocoding API to the same effect as the previous library.</p>
<p><strong>memcache-client: </strong>A pure Ruby client for memcached, the great memory cache system.</p>
<p><strong>yahoo-search: </strong>Bindings to Yahoo's search API.</p>
</blockquote>
