<p>The <a href="http://www.euruko2008.org/">registration process for the Euruko 2008 European Ruby Conference</a> has opened! It's on March 29 - 30 in Prague, Czech Republic, and costs a mere 20 Euro ($30 / £15). If you missed <a href="european-rubyists-come-to-euruko-2008-prague-march-29-30-708.html">my previous post about it</a> and want to know why you should go (especially if you're in Europe), <a href="european-rubyists-come-to-euruko-2008-prague-march-29-30-708.html">check it out</a>! I'm not too familiar with the schedule so far, but I know I'll be there, along with the famous <a href="http://drnicwilliams.com/">Dr. Nic</a> and <a href="http://www.jaikoo.com/">Jonathan Conway</a>, and I've heard word a few Japanese Rubyists will be making it over. More details as I get them! Supposedly there were quite a few UK Rubyists there last year too, so I'm looking forward to seeing a lot of British faces too.</p>
