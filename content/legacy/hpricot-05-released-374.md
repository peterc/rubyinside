<p>I was just about to go to bed but a Why release is too good to miss. Why The Lucky Stiff has <a href="http://redhanded.hobix.com/inspect/andNowHpricot05IsYours.html">announced the immediate release of Hpricot 0.5</a>. For those who haven't had the good fortune of getting down and dirty with <a href="http://code.whytheluckystiff.net/hpricot/">Hpricot</a> yet, it's basically a killer HTML parsing library for Ruby. It lets you use magic like CSS notation and XPath to hack away at HTML documents (and XML generally, if you're willing).</p>
<p>If you're already a fan, just do this to get on to the new version: <em>gem install hpricot --source http://code.whytheluckystiff.net/</em></p>
