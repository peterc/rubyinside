<p><a href="http://railsify.com/">Railsify.com</a> is a new site, developed by Adam Cooke, that acts as a central resource for Rails plugins and related tools (Rake tasks, toolkits, etc). Adam has a <a href="http://www.plasticrain.com/2007/8/20/introducing-railsify">blog post with more background and details</a>.</p>
<p>This isn't the first such site. Agile Web Development has had <a href="http://agilewebdevelopment.com/plugins">a plugins directory</a> for a while now, but Railsify is impeccably presented, has its own domain, and has even become part of a nascent "network" of Rails sites along with <a href="http://railsforum.com/">RailsForum.com</a> and <a href="http://railswork.com/">RailsWork.com</a>.</p>
