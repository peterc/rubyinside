<p><strong><a href="http://www.apress.com/book/bookDisplay.html?bID=10288">Pro ActiveRecord</a></strong> (previously known as "Pro ActiveRecord for Ruby") by <em>Kevin Marshall</em>, <em>Chad Pytel</em>, and <em>Jon Yurek</em> is an advanced-level book about using the ORM database library used primarily by Ruby on Rails. To be published on September 10, 2007, the book is now ready to <a href="http://www.amazon.com/Pro-ActiveRecord-Ruby-Databases-Rails/dp/1590598474">pre-order from Amazon.com</a> for $26.39 (non affiliate link) and runs at 300 pages, so it's likely to be a reasonably complete reference to using the library whether with Rails or not.</p>
