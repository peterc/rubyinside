<p><a href="http://deplate.sourceforge.net/">Deplate</a> is a Ruby-based tool written by <a href="http://rubyforge.org/users/tlink/">Thomas Link</a> to convert documents written in wiki-like markup to various output formats. Supported output formats include LaTeX, HTML, "HTML slides", <a title="DocBook is an XML vocabulary that lets you create documents in a presentation-neutral form that captures the logical structure of your content." href="http://www.docbook.org/">DocBook</a> (which means it can also convert to PDF, and other DocBook-supported output), and plain text.</p>
<p>In addition to being capable of reading various Wiki markup variants, a particularly interesting feature is it also handles <a title="Documentation markup for Ruby source code." href="http://rdoc.sourceforge.net/">rdoc</a> input which is very handy for generating print-ready documentation of your Ruby projects.</p>
<p>To see the results for yourself, <a href="http://deplate.sourceforge.net/Documentation.php">various samples</a> of Deplate generated output are available for your viewing pleasure. To get it, you can download the stable version from <a href="http://sourceforge.net/project/platformdownload.php?group_id=108085">Deplate project files page</a> or check out its <a href="http://deplate.sourceforge.net/Development.php">latest development version</a> from CVS.</p>
