<p>It's not a common pursuit but some developers would like to look through the source code to Ruby itself. It's a great way to start if you ultimately want to get involved with helping develop Ruby itself, but it's heavy stuff. Mauricio Fernandez, however, has put together <a href="http://eigenclass.org/hiki.rb?ruby+internals+guide#l3">a brief guide to the different source files you should read</a> relating to different elements of functionality within Ruby. Experts only.</p>
