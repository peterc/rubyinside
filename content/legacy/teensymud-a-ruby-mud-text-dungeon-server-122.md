<p><a href="http://teensymud.kicks-ass.org/wiki/show/HomePage">TeensyMud</a> is a simple MUD (multi-user dungeon - a bit like a text adventure) server written in Ruby. It's not particularly complex, but it has a great structure and there's a lot to learn from looking at the code (patterns are well used). It's also ripe for extending and for building, perhaps, a proper game or full multi-user environment on.</p>
