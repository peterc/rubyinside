<p>Here's the regular update of the most interesting stories posted on <a href="http://www.rubyflow.com/">RubyFlow</a> (a community-driven Ruby news sister site to Ruby Inside) in the past couple of weeks:</p>
<div style="margin-left: 1.5em">
<p><strong>Markdown - 59 times faster:</strong> Ryan Tomayko wants to "move past BlueCloth." The result is <a href="http://tomayko.com/writings/ruby-markdown-libraries-real-cheap-for-you-two-for-price-of-one">two significantly faster Markdown libraries</a> for Rubyists.</p>
<p><strong>Cry:</strong> <a href="http://larrytheliquid.com/2008/06/05/learn-to-cry-by-writing-parse-trees-in-ruby">Cry</a> is a Ruby library that provides a nice object oriented way to create, transfer, and manipulate frozen parse trees.</p>
<p><strong>New Ruby Social Network:</strong> <a href="http://www.actsascommunity.com/">Acts As Community</a> is a new social network for Rubyists.</p>
<p><strong>Building an MP3 Player with Ruby and Shoes:</strong> Satoshi Asakawa has put together <a href="http://rubylearning.com/blog/2008/05/31/a-teeny-weeny-mp3-player-using-ruby-and-shoes/">a cute tutorial demonstrating how to create a GUI-based MP3 player</a> using Ruby and <em>_why</em>'s Shoes library.</p>
<p><strong>Capistrano 2.4.0:</strong> Deployment tool <a href="http://weblog.jamisbuck.org/2008/6/13/capistrano-2-4-0">Capistrano has hit version 2.4.0.</a> Lots of new features and tweaks.</p>
<p><strong>"Run Ruby Script" Action for OS X's Automator:</strong> Jason Foreman has put together <a href="http://threeve.org/blog/2008/01/run-ruby-script-automator-action.html">a custom action for OS X's Automator tool</a> that makes it easy to run Ruby code from an Automator workflow.</p>
<p><strong>Lazy evaluation for Ruby methods:</strong> <a href="http://blog.astrails.com/post/37144206/being-lazy-with-ruby">Lazyeval</a> is an interesting library that makes it possible for methods to only be executed on objects when they're actually needed (ideal if you're doing caching on Rails views but still need logic in the controller).</p>
<p><strong>Guide to using Sphinx:</strong> Pat Allan has written a <a href="http://freelancing-gods.com/posts/a_concise_guide_to_using_thinking_sphinx">great article on installing and using Sphinx</a>, the search library.</p>
<p><a href="http://stephencelis.com/archive/2008/6/bashfully-yours-gem-shortcuts"><strong>Bashfully Yours, Gem Shortcuts</strong></a><strong>:</strong> A little Bash script that lets you gain quick access to the RDocs for your installed gems.</p>
<p><strong>Phusion Passenger 2.0 RC 1 and Ruby Enterprise Edition Released:</strong> The guys at Phusion have unveiled <a href="http://blog.phusion.nl/2008/06/09/phusion-passenger-20-rc-1-and-ruby-enterprise-edition-released/">a release candidate of Passenger 2.0</a> (now supporting Rack) as well as <a href="http://www.rubyenterpriseedition.com/">Ruby Enterprise Edition</a>.</p>
<p><strong>Sequel 2.0:</strong> A significant update to the Sequel database access abstraction library <a href="http://blade.nagaokaut.ac.jp/cgi-bin/scat.rb/ruby/ruby-talk/303681">has been released</a>.</p>
</div>
