<p>Looking for a Ruby/Rails job in Texas, California, Florida, Illinois, or Texas? We might have want you want in this post. This is perhaps the most diverse set of states featured in a Ruby Inside job post - usually it's just California and New York! It seems Ruby and Rails are now finding a full-time billing in a whole new set of companies.</p>
<h3>
<span style="font-size: 12px; font-weight: normal;"><img src="/assets/2009/09/equitymetrix.png" width="137" height="40" alt="equitymetrix.png" style="float:right; margin-bottom:12px; margin-left:12px;"></span><a href="http://ruby.jobamatic.com/a/jbb/job-details/127555">Ruby Application Developer</a> - EquityMetrix LLC - Dallas, Texas</h3>
<p>EquityMetrix LLC, a Dallas based business services company, is <a href="http://ruby.jobamatic.com/a/jbb/job-details/127555">looking for a Ruby application developer</a> to work alongside experienced developers building applications for internal and client use. You'll be working with Ruby, Linux, Postgresql and other open source software. Your job will consist of developing Web applications, libraries, scripts, and building data mining systems. You need a working knowledge of Ruby, Rails and JavaScript, as well as Linux and source control (svn or git).</p>

<h3>
<a href="http://ruby.jobamatic.com/a/jbb/job-details/120519">Rails Engineer (for games)</a> - Serious Business - San Francisco, California</h3>
<p><a href="http://www.seriousbusiness.com/">Serious Business</a> is a social games developer located in downtown San Francisco and they're <a href="http://ruby.jobamatic.com/a/jbb/job-details/120519">looking for an experienced Rails developer</a> to not only build cutting edge Web interfaces, but to work on their server side game logic, internal tools, and database systems. Located in San Francisco and with a staff of 19, this is an opportunity to work with a close knit team on cool stuff. Perks include 30" monitors, Aeron chairs, MacBook Pros, and full health, vision and dental benefits.</p>

<a href="http://ruby.jobamatic.com/a/jbb/job-details/113702">Rails Savvy Software Engineer</a> - Spatial Networks, Inc - Clearwater, Florida</h3>
<p>Spatial Networks, Inc is <a href="http://ruby.jobamatic.com/a/jbb/job-details/113702">looking for a challenge-seeking quick learner</a> with a few years' Rails experience to work in its Clearwater, FL office. Bonus points for familiarity with Amazon Web Services, HAML/SASS, and testing. <b><i>Note: This job listing expires on Friday, September 11, 2009 - so act fast!</i></b></p>

<h3>
<a href="http://ruby.jobamatic.com/a/jbb/job-details/114497">Ruby Software Developer</a> - Braintree Payment Solutions - Chicago, Illinois</h3>
<p>Braintree Payment Solutions <a href="http://ruby.jobamatic.com/a/jbb/job-details/114497">wants a developer</a> to help do greenfield development in the payment solutions space. You'll be working from their Chicago, IL office in a new, highly collaborative team which you'll help to define. Rails will make up most of the work to start with but they want to leverage other technologies over time, such as Erlang and Clojure, so an experimental nature would come in handy. <b><i>Note: This job listing expires on Sunday, September 13, 2009 - so act fast!</i></b></p>

<a href="http://ruby.jobamatic.com/a/jbb/job-details/128243">Ruby Application Developer</a> - The Scruggs Group - Dallas/Fort Worth, Texas</h3>
<p>The Scruggs Group, a Texas based data mining consultancy, is <a href="http://ruby.jobamatic.com/a/jbb/job-details/128243">looking for an experienced Ruby and Rails developer</a> to work full time in the Dallas / Fort Worth area.
