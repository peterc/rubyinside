<p>In <a href="http://www.railsjedi.com/posts/21-Fluid-Rails-Docs-on-OSX">Fluid Rails Docs on OSX</a>, Jacques Crocker takes a look at a couple of ways to get good-looking Rails documentation within Mac OS X. Firstly, he provides a site specific browser for <a href="http://www.noobkit.com/">NoobKit</a>, an online Rails / Gems documentation site. Secondly, however, is an app that accesses a locally available set of documentation called "<a href="http://www.railsbrain.com/">RailsBrain</a>" that works offline.</p>
