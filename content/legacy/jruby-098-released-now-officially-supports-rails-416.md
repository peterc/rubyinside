<p>On behalf of the whole JRuby team, Thomas Enebo has <a href="http://docs.codehaus.org/display/JRUBY/2007/03/05/JRuby+0.9.8+Released">announced the release</a> of <a href="http://dist.codehaus.org/jruby/">JRuby 0.9.8</a>. Hundreds of minor tweaks, fixes, and features have been added, and now <a href="http://www.headius.com/jrubywiki/index.php/Rails_Support">Ruby on Rails is officially supported</a>. The team report a 98% pass rate for Rails 1.2.1's own unit tests, and if you've seen how deep the Rails tests go, that's pretty good work. It's pretty clear JRuby isn't too far from a 1.0 release now from which full compatibility should be available.</p>
