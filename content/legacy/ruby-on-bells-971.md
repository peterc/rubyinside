<p><a href="http://vimeo.com/1261369">Ruby on Bells</a> is a demo of the <a href="http://github.com/madrona/rad/tree/master">Madrona fork</a> of <a href="http://rad.rubyforge.org/">RAD</a> (Ruby Arduino Development) itself a Ruby bridge to <a href="http://www.arduino.cc/">Arduino</a>, an open source electronics prototyping platform. Powered by a simple Ruby script, a set of glasses are hit by small mallets on servos, resulting in a Philip Glass-esque cacophony. It's a compelling demo, and if physical computing intrigues you, the tools available to Ruby and Arduino developers are now mature enough to bring interesting results within easy reach.</p>
<p>A <a href="http://vimeo.com/1272402">longer "behind the scenes" video explaining how it was put together</a> and how the code looks is also available. The code used is a very simple Ruby-based DSL. Couple this with <a href="http://gilesgoatboy.org/ruby/midi.html">Giles Bowkett's MIDI generator</a> and you could have a lot of fun.</p>
