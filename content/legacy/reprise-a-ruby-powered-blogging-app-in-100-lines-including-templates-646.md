<p><img src="/assets/2007/11/reprise.jpg" height="174" width="232" border="1" hspace="4" vspace="4" alt="Reprise"><br>
Eivind Uggedal has written in with a note about a new blog system he's built using the lightweight <a href="http://sinatra.rubyforge.org/">Sinatra</a> framework. <a href="http://redflavor.com/reprise.rb">It's about 100 lines long</a> including the inline templates! An example of the finished software powers <a href="http://journal.redflavor.com/">Eivind's own blog</a>. Sinatra is looking to be an absolutely excellent mini-framework for running up basic non-MVC applications, and you can learn more about it <a href="http://sinatra.rubyforge.org/">here</a>.</p>
