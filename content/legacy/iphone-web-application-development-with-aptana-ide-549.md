<p>The freely available <a href="http://www.aptana.com/">Aptana IDE</a> seems to be moving faster than ever. Aptana is a web developer's IDE with extensive Ruby and Rails support through the <a href="http://www.aptana.com/download_rails_rdt.php">RadRails plugin</a>. Having announced <a href="adobe-integrated-runtime-air-resources-roundup-for-ruby-on-rails-developers-531.html">support for AIR</a> right after Adobe released a Beta, the latest Aptana IDE already <a title="Aptana IDE adds support for Apple iPhone" href="http://www.aptana.com/blog/?p=152">supports development for the just-released Apple iPhone</a>.</p>
<p>Among the features that Aptana IDE provides for iPhone developers include:</p>
<ul>
<li>Preview of your iPhone projects in hortizontal and vertical mode</li>
<li>iPhone Project contains starter files to get you going easily on your iPhones apps</li>
<li>Easy import of Ajax libraries into iPhone projects</li>
<li>Help and online documentation</li>
<li>Integrated server that enables you to serve your iPhone apps directly to your iPhone for easy testing</li>
<li>A version of Joe Hewitt’s Firebug for iPhone</li>
<li>Export wizard for bundling and deploying iPhone applications (coming soon)</li>
<li>Integrated content assist for iPhone-specific APIs (coming soon)</li>
</ul>
<p>Check out the <a href="http://www.aptana.com/iphone">Aptana for iPhone development</a> page for more information. They also provided a nice <a href="http://www.aptana.com/blog/?p=153">intro to iPhone development screencast</a> for a quick start.</p>
<p><em>Note: If you like Aptana IDE, you may want to consider joining their team, as they're currently hiring developers.</em></p>
