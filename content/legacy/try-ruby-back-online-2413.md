<p><a href="http://tryruby.sophrinix.com/"><img src="/assets/2009/09/screen-shot-2009-09-02-at-191938.png" width="123" height="76" alt="Screen shot 2009-09-02 at 19.19.38.png" style="float:left; margin-right:12px; margin-bottom:12px;"></a><i>Try Ruby</i> was a Web site by <a href="why-the-lucky-stiff-links-2333.html">Why The Lucky Stiff</a> that provided a Web-based version of <a href="irb-lets-bone-up-on-the-interactive-ruby-shell-1771.html">irb</a> (the <b>i</b>nteractive <b>r</b>u<b>b</b>y prompt) and a 15 minute tutorial for people to learn and play with Ruby. With <a href="why-the-lucky-stiff-is-missing-2278.html">Why's disappearance</a>, however, the site went down and an invaluable Ruby community resource was lost.</p>
<p>Luckily, Andrew McElroy has made a great effort in <a href="http://bit.ly/XJ36a">getting Try Ruby back online</a>. It's not <i>precisely</i> the same, but it's as close as you're going to get for now (one key difference is that the irb process is not persistent - instead the history is re-run on each new line). There's even the 15 minute introductory Ruby tutorial! An extra bonus is that Andrew has <a href="http://github.com/Sophrinix/TryRuby/tree/master">released all of the code he has</a> for Try Ruby in <a href="http://github.com/Sophrinix/TryRuby/tree/master">a Github repository</a>. Why's code was never open sourced but Andrew has done a good job in starting to rebuild the backend.</p>
<p><em>Note that this is <strong>not a Why-approved project</strong> (yet). To the best of my knowledge there is still no further information on Why's disappearance.</em></p>
