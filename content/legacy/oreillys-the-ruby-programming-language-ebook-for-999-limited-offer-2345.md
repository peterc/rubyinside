<p><img src="/assets/2009/08/rubyprogramminglanguage.gif" width="111" height="145" alt="rubyprogramminglanguage.gif" style="float:left; margin-right:12px; margin-bottom:12px;">I've just had word from David Flanagan - co-author of <a href="http://oreilly.com/catalog/9780596516178/">The Ruby Programming Language</a>, published by O'Reilly - that O'Reilly are running a temporary sale on the e-book edition. <b>The PDF e-book is now just..</b> <a href="http://oreilly.com/catalog/9780596516178/"><b>$9.99 for a limited time only</b></a><b>.</b> That's a pretty good deal considering O'Reilly are selling the print book for $40 (though it's only <a href="http://www.amazon.com/Ruby-Programming-Language-David-Flanagan/dp/0596516177/">$26.39 on Amazon</a>).</p>
<p><strong>Actually.. make that $5.99? Reader Mike Hodgson claims: <em>If you use the LREL40 coupon code you can get an additional $4.00 off, bringing it down to $5.99</em></strong> - Not tried it myself, but it's worth a go.</p>
<p>The Ruby Programming Language is the canonical Ruby book covering both Ruby 1.8 and 1.9. It's by David Flanagan and Yukihiro Matsumoto (Matz!) so you know it's authoritative. When <a href="the-new-most-important-ruby-book-oreillys-the-ruby-programming-language-778.html">we reviewed it in 2008</a>, we billed it "the new most important Ruby book."</p>
<p><b>David also mentions that there's</b> <a href="http://appshopper.com/books/the-ruby-programming-language"><b>a new iPhone edition available too for a mere $5.</b></a> A great opportunity to get a great book on the cheap, whatever your budget.</p>
