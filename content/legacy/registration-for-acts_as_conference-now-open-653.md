Robert Dempsey of non-profit Rails advocacy group, Rails For All, writes in to remind everyone about the <a href="http://www.actsasconference.com/">acts_as_conference Rails conference</a> taking place in Florida in February 2008 and to let us know that <a href="http://actsasconference.eventbrite.com/">registration</a> is now open. Tickets cost $100 (plus $2.50 booking fee). <a href="http://www.actsasconference.com/speakers/obie-fernandez">Obie Fernandez</a> and <a href="http://www.actsasconference.com/speakers/dan-benjamin">Dan Benjamin</a> are the keynoter speakers, but there are many others. Too many to name individually here, although <a href="http://www.actsasconference.com/speakers/charles-nutter">Charles Nutter</a> (JRuby), <a href="http://www.actsasconference.com/speakers/ezra-zygmuntowicz">Ezra Zygmuntowicz</a> (Merb), and <a href="http://www.actsasconference.com/speakers/evan-phoenix">Evan Phoenix</a> (Rubinius) are particular standouts. Anyway, if you fancy getting some winter sun while doing the Rails schmooze, hit it up. It doesn't sound like you're going to be bored at this one.</p>
