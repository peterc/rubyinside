<p>Ruby is an easy language to learn, but it's often necessary to look up something we've forgotten. A combination of Google plus any Ruby books we have on our shelves can help, but sometimes it's handy to refer to a simpler set of notes - such as a "cheat sheet." This post attempts to cover the most interesting ones.</p>
<p><em>The idea for this initial list came from Scott Klarr's <a href="http://www.scottklarr.com/topic/104/ruby-and-ruby-on-rails-cheat-sheets/">own list.</a> Scott has been quite prolific lately in putting together lists of cheat sheets. Some of his <a href="http://www.scottklarr.com/topic/109/cheat-sheet-index/">lists</a> are: <a href="http://www.scottklarr.com/topic/107/apache-cheat-sheets/">Apache cheat sheets</a>, <a href="http://www.scottklarr.com/topic/102/mysql-cheat-sheets/">MySQL cheat sheets</a>, <a href="http://www.scottklarr.com/topic/100/php-cheat-sheets/">PHP cheat sheets</a>, and <a href="http://www.scottklarr.com/topic/95/javascriptajax-cheat-sheets/">JavaScript / AJAX cheat sheets.</a></em></p>
<h3><strong>On with the Ruby cheat sheets:</strong></h3>
<h3>"<a href="http://refcardz.dzone.com/refcardz/essential-ruby">Essential Ruby</a>" RefCard (PDF)</h3>
<p><a href="http://refcardz.dzone.com/refcardz/essential-ruby"><img class="alignnone size-medium wp-image-1340" title="Ruby Refcardz" src="/assets/2008/11/refcard-1.png" alt="" width="206" height="266"></a></p>
<p><a href="http://refcardz.dzone.com/refcardz/essential-ruby">Essential Ruby</a> is a combination of a cheat sheet and a tutorial. It's six pages long, but features a mini Ruby introduction and tutorial, as well as the myriad of tables you'd expect from a cheat sheet. It's like a whole Ruby beginner's reference and tutorial in a single PDF. The only "catch" is that you need to be a user of DZone (the "Digg for Developers") or JavaLobby to get it, but there's a very simple signup form you can fill in if you're not.</p>
<p><big><strong><a href="http://ruby.cenophobie.com/RubyCheat.pdf">Ruby Cheatsheet</a> (PDF and <a href="http://ruby.cenophobie.com/rubycheat.php">PNG</a>)</strong></big></p>
<div style="margin-left: 2em">
<p><img src="/assets/2008/02/rubycheat.png" alt="rubycheat.png" width="305" height="285"></p>
<p>This cheat sheet (<a href="http://ruby.cenophobie.com/rubycheat.php">homepage</a>) covers types, exceptions, expressions, variable types, operators and precedence, constants, regular expressions, predefined / special variables, arguments accepted by the Ruby interpreter, reserved words, and a large collection of Object, String, Kernel, Array, Hash, Test::Unit, File, Dir and DateTime methods. Extremely suitable for printing and/or wall chart use.</p>
</div>
<p><big><strong><a href="http://www.zenspider.com/Languages/Ruby/QuickRef.html">Ruby QuickRef</a> (HTML)</strong></big></p>
<div style="margin-left: 2em">
<p><img src="/assets/2008/02/rubyquickref.png" alt="rubyquickref.png" width="286" height="231"></p>
<p>I often use this one myself! A ridiculously comprehensive "cheat sheet" created by Ryan Davis covering syntax rules, reserved words, regular expression terminology, class and method definitions, predefined variables and constants, control and logic expressions and formations, exceptions, the standard library, and a whole ton of useful stuff.</p>
</div>
<p><big><strong><a href="http://www.testingeducation.org/conference/wtst3_pettichord9.pdf">Ruby Reference</a> (PDF)</strong></big></p>
<div style="margin-left: 2em">
<p><img src="/assets/2008/02/rubycheatsheet.png" alt="rubycheatsheet.png" width="287" height="239"></p>
<p>More of an extremely brief reference than a truly comprehensive cheat sheet, but still useful, especially for beginners. 5 pages long, so not suitable for printing (Also <a href="http://www.scribd.com/doc/53239/Ruby-Cheat-Sheet">available</a> to view on the Web at Scribd.)</p>
</div>
<p><big><strong><a href="http://cheat.errtheblog.com/">Cheat: Ruby Cheat Sheets on your Terminal!</a> (Application)</strong></big></p>
<div style="margin-left: 2em">
<p><img src="/assets/2008/02/errcheat.png" alt="errcheat.png" width="289" height="192"></p>
<p>Not a cheat sheet, but an application (<span style="font-style: italic;">gem install cheat</span>) that provides cheat sheet type reference material on your terminal (from 228 source cheat sheets!) Developed by the great guys over at <a href="http://errfree.com/">Err Free.</a></p>
</div>
<p><big><strong><a href="http://blog.unquiet.net/ruby-cheatsheet/">A Beginner's Notes</a> (HTML)</strong></big></p>
<div style="margin-left: 2em">
<p><img src="/assets/2008/02/beginnersnotes.png" alt="beginnersnotes.png" width="284" height="246"></p>
<p>A set of notes made by a beginning Ruby developer as he works his way through a Ruby manual.</p>
</div>
<p><big><strong><a href="http://cheat.errtheblog.com/s/ruby/">Ruby Command Line Parameters Cheat Sheet</a> (HTML)</strong></big></p>
<div style="margin-left: 2em">
<p>A simplified list of the command line options offered by the Ruby interpreter.</p>
</div>
<p><big><strong><a href="http://cheat.errtheblog.com/s/strftime/">Strftime Parameters</a> (HTML)</strong></big></p>
<div style="margin-left: 2em">
<p>A handy guide to the formatting parameters accepted by strftime. I commonly have to look these up myself!</p>
</div>
