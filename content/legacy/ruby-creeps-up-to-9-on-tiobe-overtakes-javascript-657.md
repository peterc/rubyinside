<p><img src="/assets/2007/11/tiobenov2007.jpg" height="236" width="373" border="1" hspace="4" vspace="4" alt="Tiobenov2007"><br>
The <a href="http://www.tiobe.com/index.htm?tiobe_index">TIOBE "Programming Community Index" chart for November 2007</a> has been released and Ruby has climbed one place from #10 to #9, overtaking JavaScript in the process. The TIOBE chart is not a particularly accurate chart of programming language use, but it provides a data point used by many. Over the last two years we've watched Ruby scale quite a few places.</p>
