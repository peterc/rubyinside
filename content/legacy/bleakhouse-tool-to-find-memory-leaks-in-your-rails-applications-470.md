<p><a href="http://blog.evanweaver.com/articles/2007/04/28/bleak_house">Bleakhouse</a> is a new Rails plugin by Evan Weaver that helps you find memory leaks in your Rails applications. It does this by hammering ObjectSpace for information throughout the execution of your app and by producing pretty charts to show you what's going on. It's easy to install and get running, so if memory leaks or application stability have proven to be sore points for you, give it a try straight away.</p>
