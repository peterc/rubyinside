<p><strong style="font-size: 1.1em">Please note that this post is over four years old - it's from 2006! As such, these tips were relevant to Rails 1.2 and this content is woefully out of date. We will be updating it for Rails 3 soon! <p>RJS is a template type for Rails that allows Rails developers to use Ruby to produce dynamic JavaScript code that will usually be executed in response to an AJAX call. There are lot of disparate resources relating to RJS around the net, so I've collected as many of them together here as possible. If you need to use RJS, this is the list for you!</p>
<p><strong>Introduction / Tutorials</strong></p>
<ul>
<li>
<a href="http://www.rubynoob.com/articles/2006/05/13/simple-rails-rjs-tutorial">Simple RJS Tutorial</a> - An up to date RJS tutorial (May 2006) featuring a video demonstration and some code.</li>
</ul>
<ul>
<li>
<a href="http://blog.fearoffish.com/articles/2006/03/04/tutorial-rails-rjs-templates">RJS Tutorial by FearOfFish</a> - Another overall tutorial with a demonstration view, controller, and RJS template for adding a shopping cart type system to a page.</li>
</ul>
<ul>
<li>
<a href="http://scottraymond.net/articles/2005/12/01/real-world-rails-rjs-templates">Real-world Rails RJS Templates</a> - An introductory look at the benefits of RJS templates posted close to their introduction in December, 2005.</li>
</ul>
<ul>
<li>
<a href="http://www.railsweenie.com/forums/2/topics/784">Ajaxed forms with RJS templates</a> - Technoweenie demonstrates how an entire form can be processed and manipulated using AJAX and RJS. Lots of nice syntax colored code!</li>
</ul>
<ul>
<li>
<a href="http://depot.iamjp.com/demo">A Demo of RJS</a> - A live demo of what RJS can do, including code.</li>
</ul>
<ul>
<li>
<a href="http://www.bigbold.com/snippets/posts/show/1705">Redirection with RJS and AJAX</a> - Showing some code to make a login process easier with RJS.</li>
</ul>
<p><strong>References</strong></p>
<ul>
<li>
<a href="http://api.rubyonrails.org/classes/ActionView/Helpers/PrototypeHelper/JavaScriptGenerator/GeneratorMethods.html">Official RJS documentation from the Rails API docs</a> - The official docs don't usually show the latest tricks, but it's great for the basics.</li>
</ul>
<ul>
<li>
<a href="http://rewrite.rickbradley.com/articles/2006/02/06/rjs-templates">RJS Template Methods</a> - A list of methods supported by RJS templates from February 2006 (before Rails 1.1). It includes some demonstration code to put together a small app using RJS.</li>
</ul>
<ul>
<li>
<a href="http://www.nshb.net/rjs-templates-simplified.html">RJS Templates, simplified</a> - Another basic list of the RJS methods.</li>
</ul>
<p><strong>Miscellaneous / Advanced</strong></p>
<ul>
<li>
<a href="http://www.memerocket.com/2006/02/10/tell-textmate-about-your-rjs-templates/">Add RJS support to TextMate</a> - TextMate doesn't understand RJS by default, and this post shows you how to get the syntax coloring working.</li>
</ul>
<ul>
<li>
<a href="http://habtm.com/articles/2005/12/15/another-use-for-rjs-templates">Another use for RJS templates</a> - Advanced. A look at how to use the handy RJS methods from Prototype callbacks and static javascript files.</li>
</ul>
<p><strong>Debugging and Testing</strong></p>
<ul>
<li>
<a href="http://blog.vandev.com/articles/2006/05/04/my-first-tool-for-you">AjaxSpy</a> - A cute way to debug and monitor RJS requests directly from the page. Only works in FireFox and Safari.</li>
</ul>
<ul>
<li>
<a href="http://rails.techno-weenie.net/tip/2005/12/20/debugging_your_rjs_calls">Debugging your RJS calls</a> - Find out what RJS is getting returned from the page end of things with this useful JavaScript trick.</li>
</ul>
<ul>
<li>
<a href="http://blog.labnotes.org/2006/05/21/tips-for-debugging-rjs-code/">Tips for debugging RJS code</a> - A list of useful tips and considerations when debugging RJS code.</li>
</ul>
<ul>
<li>
<a href="http://ibrasten.com/articles/2006/04/05/rjs-assertions">RJS Assertions Plugin</a> - A plugin to help you with testing your RJS.</li>
</ul>
<ul>
<li>
<a href="http://www.oobaloo.co.uk/articles/2006/04/25/rjs-assertions">RJS Assertions and Testing</a> - Useful tips for testing RJS code and responses.</li>
</ul>
