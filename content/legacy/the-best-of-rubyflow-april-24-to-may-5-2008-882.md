<p><a href="http://www.rubyflow.com/">RubyFlow</a> - the community based companion site to Ruby Inside - has been on fire! I'm finding out about lots of new stuff on there that then gets included into Ruby Inside posts. It's the place to be if you want the most up to date Ruby and Rails news, but don't mind putting up with a bit of 'noise'.</p>
<p>Every two weeks or so I'm going to summarize some of the best items from RubyFlow here on Ruby Inside, so that you can still keep up with the latest developments even if you don't want to be soaked in the firehose of Ruby news over there.</p>
<p>For the period April 24 to May 5, 2008:</p>
<p><strong>Net::SSH 2.0 Released:</strong> Jamis Buck <a href="http://weblog.jamisbuck.org/2008/5/2/net-ssh-2-0-and-friends">announces the release of Net::SSH 2.0</a> and the availability of Net::SFTP 2.0, Net::SCP 1.0, Net::SSH::Gateway 1.0 and Net::SSH::Multi 1.0.</p>
<p><strong>Webistrano 1.3:</strong> <a href="http://labs.peritor.com/webistrano">Webistrano</a> 1.3 has been released; read <a href="http://blog.innerewut.de/2008/4/29/webistrano-1-3-released">the announcement</a>. Webistrano is a Web UI for managing Capistrano deployments. It lets you manage projects and their stages like test, production, and staging with different settings. Those stages can then be deployed with Capistrano through Webistrano.</p>
<p><strong>Automatic Migration Generator:</strong> <a href="http://hobocentral.net/hobofields/">Hobofields</a> is an automatic migration generator for Rails / ActiveRecord users. Annotate your model with the fields required as you go, then Hobofields generates the required migrations.</p>
<p><strong>Capistrano 2.3.0:</strong> Yehey! <a href="http://weblog.jamisbuck.org/2008/5/2/capistrano-2-3-0">Capistrano 2.3.0</a> is released. It has many new tasty features!</p>
<p><strong>Rails 2.1 Features:</strong> A summary of some of the <a href="http://railspikes.com/2008/5/2/rails-gets-more-mature" rel="nofollow">nice new features coming in Rails 2.1</a>. In short, many of the rough spots are being patched over!</p>
<p><strong>Ruby and TextMate:</strong> An interesting <a href="http://ciaranwal.sh/2008/02/20/textmate-tip-the-ruby-bundle">introduction to TextMate's Ruby bundle.</a> A good place to start if you use TextMate but haven't used any of the mnemonics and snippets the Ruby bundle provides (like me).</p>
<p><strong>John Lam on Iron Ruby:</strong> A <a href="http://codebetter.com/blogs/david_laribee/archive/2008/04/24/john-lam-on-ironruby.aspx">video</a> update on Iron Ruby from John Lam recorded by <a href="http://codebetter.com/blogs/david_laribee/default.aspx">David Laribee</a>.</p>
<p><strong>MetricFu:</strong> Jake Scruggs <a href="http://jakescruggs.blogspot.com/2008/04/dead-simple-rails-metrics-with-metricfu.html">demonstrates how to use MetricFu</a> to produce good looking metrics and reporting for your Rails application.</p>
<p><strong>Merb Blogging Software:</strong> <a href="http://crazycool.co.uk/2008/04/26/announcing-feather">Announcing Feather</a>, a Merb based blogging engine with a lightweight core framework, and a robust set of plugins, now open source and ready for contributions!</p>
<p><strong>Parsing Quoted Strings:</strong> If you need to <a href="http://mikenaberezny.com/2008/04/28/parsing-quoted-strings-in-ruby/">parse quoted strings</a> in Ruby, a lesser-known module called Shellwords from the Ruby Standard Library is a handy utility.</p>
