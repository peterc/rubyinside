<p>Just over a month ago I <a href="playing-catchup-5-great-links-from-this-week-568.html">mentioned the release of "Shoes"</a>, a new Ruby library for creating "Web-like desktop apps" by Why The Lucky Stiff. <a href="http://code.whytheluckystiff.net/shoes/">Shoes</a> provides a somewhat platform-independent way of creating desktop applications easily. Now that Shoes is several weeks old, it's worth pointing to a few resources and tutorials that will come in useful if you decide to start building Shoes-powered apps:</p>
<p><strong><a href="http://www.juixe.com/techknow/index.php/2007/08/27/running-with-shoes-a-mini-gui/">Running with Shoes - A Mini GUI Toolkit</a></strong> by <em>Juixe</em>: A great walkthrough building a simple Shoes application.</p>
<p><strong><a href="http://hackety.org/2007/08/28/cutHolesInShoesAndGetAMask.html">Cut Holes in Shoes and Get A Mask</a></strong><strong> </strong>by<strong> </strong><em>Why The Lucky Stiff</em>: A demonstration of using masks to create fancy text graphics.</p>
<p><strong><a href="http://www.juixe.com/techknow/index.php/2007/08/27/running-with-shoes-2d-graphics-and-animation/">Running with Shoes - 2D Graphics and Animation</a></strong> by <em>Juixe</em>: A look at drawing shapes in a Shoes app, such as lines, ovals, and rectangles, along with simple animation techniques.</p>
<p><strong><a href="http://code.whytheluckystiff.net/svn/shoes/trunk/samples/edit.rb">Source code for a basic text editor</a></strong><strong> </strong>by <em>Why The Lucky Stiff</em>: Source for a bare bones editing tool.</p>
<p><strong><a href="http://hackety.org/2007/08/06/yeahShoesHasJustAFewSimpleControls.html">Yeah, Shoes Has Just a Few Simple Controls</a></strong> by <em>Why The Lucky Stiff</em>: A demonstration of a few of Shoes' widgets.</p>
<p><strong><a href="http://hackety.org/2007/08/02/mashingInSomeGraphics.html">Mashing In Some Graphics</a></strong> by <em>Why The Lucky Stiff</em>: A code example of drawing rectangles within a Shoes application. </p>
<p><strong><a href="http://hackety.org/2007/08/17/hypershoes.html">Hypershoes</a></strong> by <em>Why The Lucky Stiff</em>: A look at how to build a "book reader" program in Shoes and how Shoes' concept of URLs helps.</p>
