<p><a href="http://www.plasmacode.com/">RoRED</a> is a new Ruby / Rails IDE for Windows by Marcus Oblak.</p>
<p>I don't run Windows, but it appears to have easy hooks for the dev server, console, and breakpointer built in, as well as easy ways to navigate through methods and models. It also features macro recording and playback, bookmarking, persistent layouts, and Ruby and RHTML syntax highlighting.</p>
