<p><a href="http://www.flickr.com/photos/pdcawley/252275180/"><img src="/assets/2009/08/whymo.png" width="104" height="117" alt="whymo.png" style="float:right; margin-bottom:12px; margin-left:12px; border:1px #000000 solid;"></a>Since Why The Lucky Stiff's <a href="why-the-lucky-stiff-is-missing-2278.html">supposed disappearance yesterday</a>, I've been coming across a lot of interesting links with mirrors of his projects, discussions surrounding his disappearance, and lots of other juicy stuff. I think it's worth keeping track of this stuff so I'm kicking off a compilation post dedicated to Why related links.</p>
<p>If you have any useful resources you want to link to, leave a comment and I'll try and include them here so this post can act as a reference point.</p>
<h3>Disappearance and Discussion</h3>
<p><a href="http://www.reddit.com/r/programming/comments/9c5on/where_is_why/">Where is _why? A Reddit discussion</a> - An epic discussion (443 comments at time of writing) on Reddit about Why's disappearance. The top moderated comment is a little mean: "Why couldn't DHH disappear instead?" Nevertheless, some interesting points come up, including a bit of detective work that resolves whytheluckystiff.net's IP address to the same as that for <a href="http://cewki.com/">cewki.com</a> which asks where Why is (<a href="http://blog.hamptoncatlin.com/2009/where-is-_why">Hampton Catlin has a roundup</a> of this line of inquiry).</p>
<p><a href="http://whoiswhytheluckystiff.wordpress.com/">Who Is Why The Lucky Stiff?</a> - A site that supposedly reveals Why's true identity as "Jonathan Gillette" due to headers in ruby-talk mailings.. but I'm not buying it. A lot of commentators have blamed this site for Why's disappearance though. It's not known who put this site up.</p>
<p><a href="http://news.ycombinator.com/item?id=773106">Hacker News: _why is no more</a> - The Hacker News and discussion of Why's disappearance. Somewhat more mature and well meaning than the Reddit discussion.</p>
<p><a href="http://www.metafilter.com/84292/Where-and-why">Where and _why</a> - A discussion about Why's disappearance on MetaFilter (of which Why was a reader). It eventually turns into a discussion about how C# is or isn't turning into Ruby. There's also <a href="http://www.metafilter.com/34182/if-programming-books-were-written-by-crazy-people">a thread from July 2004</a> when Why's book was featured there.</p>
<p><a href="http://www.h-online.com/open/why-the-lucky-stiff-Vanishes--/news/114042">"why the lucky stiff" Vanishes</a> - Heise Online's coverage of Why's disappearance.</p>
<p><span style="font-size: 14px; font-weight: bold;">Why's Code and Contributions</span></p>
<p><a href="http://github.com/whymirror">A GitHub mirror of Why's projects</a> - A user called "whymirror" has been formed on GitHub and features forks and mirrors of Why's projects, including <a href="http://github.com/whymirror/shoes/tree">Shoes</a>, <a href="http://github.com/whymirror/poignantguide/tree">The Poignant Guide</a>, <a href="http://github.com/whymirror/camping/tree">Camping</a>, and <a href="http://github.com/whymirror/hpricot/tree">Hpricot</a>.</p>
<p><a href="http://favstar.fm/users/_why">Why's most favorited Twitter messages</a> - favstar.fm collates the "most favorited" tweets for Twitter users, and this is a compilation of Why's top tweets. It includes some classics:</p>
<blockquote>
<p>trying to reading dhh’s articles on himself, but his website is so drenched in axe body spray that it has more of a tear gas effect.</p>
</blockquote>
<blockquote>
<p>when you don't create things, you become defined by your tastes rather than ability. your tastes only narrow &amp; exclude people. so create.</p>
</blockquote>
<blockquote>
<p>if you program and want any longevity to your work, make a game. all else recycles, but people rewrite architectures to keep games alive.</p>
</blockquote>
<p><a href="http://mislav.uniqpath.com/poignant-guide/">An HTML mirror of The Poignant Guide</a> - Courtesy of GitHub Pages, the Poignant Guide is back online in all its former glory.</p>
<p><a href="/assets/media/poignant-guide.pdf">A PDF mirror of The Poignant Guide</a> - We've put up a mirror of a PDF version of Why's famous <i>Poignant Guide</i>.</p>
<p><a href="http://www.vimeo.com/5047563">ART &amp;&amp; CODE Symposium: Hackety Hack</a> - A recent(ish) presentation by Why at the ART &amp;&amp; CODE Symposium. This is a great way to see him "in the flesh."</p>
<p><a href="http://web.archive.org/web/20071018035725/poignantguide.net/sdtrk/">The soundtrack to why's (poignant) guide to ruby</a> - archive.org has a mirror of the soundtrack to the Poignant Guide as performed by Why's band, The Thirsty Cups. 8 MP3s. I find archive.org to be frequently unreliable so if you like these you might want to mirror them yourself.</p>
<p><a href="http://1.bp.blogspot.com/_UZXge-Hh5UU/R1PG_OXr_JI/AAAAAAAAABo/FiMy--Jltgc/s1600-R/tryruby.jpg">A screenshot of tryruby.hobix.com</a> - tryruby.hobix.com was a Web version of irb, the interactive Ruby interpreter. While it's down, <a href="http://codepad.org/">codepad.org</a> might be useful, though it's not really the same thing.</p>
<p><a href="http://the-shoebox.org/">The Shoebox</a> - A repository of Shoes applications. This Web site was not owned by Why, hence it's still up.</p>

<p><a href="https://www.youtube.com/watch?v=_EPzGpwnekg&amp;feature=related">Why the Lucky Stiff on Twins and Drawing</a> - A video made to some spoken word by Why. A couple of quirky stories.</p>
<p><a href="https://www.youtube.com/watch?v=YReSQfSBz4k">The Belljar, the Space, and My Daughter's Organ Instructor</a> - A commemorative video by "Raymond" set to Why's music.</p>
