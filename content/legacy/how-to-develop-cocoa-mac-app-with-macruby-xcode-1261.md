<p><img src="/assets/2008/10/macrubytutorial.png" width="147" height="118" alt="macrubytutorial.png" style="float:left; margin-right:12px; margin-bottom:12px;">Over a year ago we had a post about <a href="how-to-build-os-x-gui-applications-with-ruby-and-rubycocoa-514.html">how to build OS X GUI applications with Ruby and RubyCocoa.</a> Since then, however, <a href="http://www.macruby.org/trac/wiki/MacRuby">MacRuby</a> has arrived on the scene. Not just the regular version of Ruby with some bindings to Cocoa, MacRuby is <a href="http://www.macruby.org/trac/wiki/WhyMacRuby">as native to OS X</a> as JRuby is native to the JVM.</p>
<p><strong>It's a significant development, then, that Apple has created a very in-depth tutorial called</strong> <a href="http://developer.apple.com/mac/articles/scriptingautomation/cocoaappswithmacruby.html"><strong>Developing Cocoa Applications Using MacRuby</strong></a><strong>.</strong> It's very comprehensive. The tutorial introduces you to MacRuby, walks you through installation, demonstrates Objective-C bindings, and steps through using XCode and MacRuby to develop a simple GUI application. If you want to read a single tutorial and be able to develop a Mac GUI app with Ruby from start to finish, <a href="http://developer.apple.com/mac/articles/scriptingautomation/cocoaappswithmacruby.html">this</a> is currently <em>the</em> tutorial to read.</p>
<p>That said, you might still appreciate our <a href="the-ultimate-list-of-rubycocoa-tutorials-tips-and-tools-728.html">Ultimate List of RubyCocoa Tutorials, Tips, and Tools</a>. There are 42 links there; many of which are still incredibly useful.</p>
