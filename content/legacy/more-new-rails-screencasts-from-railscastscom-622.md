<p>Ryan Bates is being a total champ in rolling out more and more consistently good Rails related screencasts for free at <a href="http://railscasts.com/">RailsCasts.com</a>. Some of the latest include:</p>
<blockquote>
<p><a href="http://railscasts.com/episodes/72">Adding an Environment</a> - Rails comes with three environments: development, test, and production. But, you aren't restricted to just these. You can add your own! See how in this episode.</p>
<p><a href="http://railscasts.com/episodes/71">Testing Controllers</a> - Controllers are tricky to test, and there's no perfect way to do it. In this episode you will see how I test controllers, and my reasoning behind it.</p>
<p><a href="http://railscasts.com/episodes/70">Custom Routes</a> - In this episode you will learn how to add custom routes, make some parameters optional, and add requirements for other parameters.</p>
<p><a href="http://railscasts.com/episodes/69">Markaby in Helper</a> - Do you ever need to generate HTML code in a helper method? Placing it directly in Ruby strings is not very pretty. Learn a great way to generate HTML through Markaby in this episode.</p>
<p><a href="http://railscasts.com/episodes/68">OpenID Authentication</a> - Ever wonder how to implement OpenID authentication in your Rails app? This episode will show you how to add it to a site with an existing authentication system.</p>
<p><a href="http://railscasts.com/episodes/67">restful_authentication</a> - Need multiple user authentication? If so, the restful_authentication plugin is a great way to go. It will generate some basic authentication code for you which is good starting point to your authentication system. Watch this episode for details.</p>
<p><a href="http://railscasts.com/episodes/66">Custom Rake Tasks</a> - Rake is one of those tools that you don't realize how powerful it is until you start using it. In this episode you will learn how to create custom rake tasks and improve them by using rake features.</p>
<p><a href="http://railscasts.com/episodes/65">Stopping Spam with Akismet</a> - The Railscasts site has been getting a lot of comment spam in the past, but no longer. In this episode I will show you how I solved this problem by using the Akismet web service.</p>
</blockquote>
<p>Keep up the great work Ryan! And if you find the videos useful in any way, Ryan is taking donations with a PayPal button on the right side of <a href="http://railscasts.com/">the homepage</a>.</p>
