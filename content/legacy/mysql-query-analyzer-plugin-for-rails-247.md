<p>Bob Silva has put together a fine <a href="http://www.railtie.net/articles/2006/09/20/mysql-query-analyzer-rails-plugin">MySQL query analyzer plugin for Ruby on Rails</a>. It uses MySQL's EXPLAIN command to analyze all of the different SQL queries made by your Rails application (or, more accurately, by ActiveRecord!) and gives you the information you need to add better indexes (or add any extra indexes at all!) and could help show where a little custom SQL could silence a squeaky wheel.</p>
