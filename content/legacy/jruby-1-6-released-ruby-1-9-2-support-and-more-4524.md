<p><img src="/assets/2011/03/jruby16.gif" alt="" title="jruby16" width="100" height="100" class="alignnone size-full wp-image-4525" style="float: left; margin-right: 18px; border: 1px solid #666">It's a newsflash! <strong><a href="http://jruby.org/2011/03/15/jruby-1-6-0.html">JRuby 1.6.0 has been released</a> today.</strong> Congratulations to the <a href="http://jruby.org/">JRuby</a> team. 1.6 is a significant and much awaited release and comes after a 9 month push of over 2500 commits.</p>
<p>Hit up <a href="http://jruby.org/2011/03/15/jruby-1-6-0.html">the official release post</a> for the full run-through but here are some of the highlights of the release:</p>
<ul>
<li>Windows has been added to the JRuby team's continuous integration system meaning that Windows support is only going to get better</li>
<li>Ruby 1.9.2 language and API support (with the exception of <code>Encoding::Converter</code> and <code>ripper</code>)</li>
<li>Built in profiler</li>
<li>General performance improvements</li>
<li>Experimental support for C extensions (with provisos)</li>
<li>RSpec is no longer included (worth mentioning in case it catches you out..)</li>
</ul>
<p>You can <a href="http://www.jruby.org/download">download binary and source releases</a> direct from JRuby.org if you want to get up to date or update <a href="http://rvm.beginrescueend.com/">RVM</a> with <code>rvm get head</code> and <code>rvm reload</code> before running <code>rvm install jruby-1.6.0</code> :-)</p>
<p>Fingers crossed for some great JRuby tutorials and guides coming along in the next couple of months.</p>
