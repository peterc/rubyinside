<blockquote><p><em><a href="http://narray.rubyforge.org/">NArray</a></em><em> is an Numerical N-dimensional Array class. Supported element types are 1/2/4-byte Integer, single/double-precision Real/Complex, and Ruby Object. This extension library incorporates fast calculation and easy manipulation of large numerical arrays into the Ruby language.</em></p></blockquote>
<p>It's impressive stuff, if you can understand the math. NArray lets you create arrays with any number of dimensions, and gives you methods that let you perform complex matrix math. There are some great demonstrations, including <a href="http://narray.rubyforge.org/demo/image.html.en">image resizing</a> and <a href="http://narray.rubyforge.org/demo/smooth.html.en">image smoothing</a> with the class. Or how about <a href="http://narray.rubyforge.org/demo/lifegame.html.en">the Game of Life</a> in under 40 lines of code?</p>
