<p><img src="/assets/2007/09/anvil1.jpg" height="188" width="255" border="1" hspace="4" vspace="4" alt="Anvil1"><br>
<a href="http://anvil.rubyforge.org/">Anvil</a> is a new Ruby framework for developing GUI applications by Lance Carlson. It's a framework around Wx::Ruby (<a href="http://www.wxwidgets.org/">WxWidgets</a> is a popular cross-platform widget toolkit) and offers its own DSL to make developing GUI applications easy. The <a href="http://grasprubyonrails.com/2007/8/13/anvil-the-ruby-application-framework">release blog post</a> has more information, including a great example of the code required to make a basic Anvil app. Interestingly it supports <a href="http://grasprubyonrails.com/2007/8/16/mvc-support-and-view-rendering-in-anvil">an MVC style of development</a> where the view and control elements of your applications can be separated cleanly.</p>
<p>The first Anvil-powered application is a Ruby based editor called <a href="http://hamr.rubyforge.org/">Hamr</a>. Lance is so far the main developer of the Anvil project but he's keen to get other people involved.</p>
