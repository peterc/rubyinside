<p>A lot of people have considered developing a content management system in Rails over the past year, but few attempts have turned into something usable. Enter <a href="http://www.radiantcms.com/">Radiant CMS</a>. It's not finished yet, but it provides a good example of a Rails application and is <a href="http://dev.radiantcms.org/">available now via SVN</a>. If digging through Rails source code is the way you learn best, enjoy!</p>
