<p>I don't know much about Oracle, but many people have asked about Ruby's support for Oracle. One of my clients is also attempting a project with Rails and Oracle. Here are some useful resources I've found:</p>
<p><a href="http://www.it-eye.nl/weblog/2006/01/06/starting-with-ruby-and-oracle/">Starting with Ruby and Oracle</a> : A pretty comprehensive and up to date tutorial about using Oracle with Ruby. The tutorial is written from a Windows perspective.</p>
<p><a href="http://www.jiubao.org/ruby-oci8/index.en.html">Ruby/OCI8 (Oracle Call Interface)</a> : A library that provides an interface between Ruby and Oracle 8.</p>
<p><a href="http://www.yoshidam.net/Ruby.html#oracle">Ruby Oracle 7 library</a> : An alternative for Oracle 7 users.</p>
<p><a href="http://www.alfonsomartone.itb.it/xvpucf.html">Ruby and Oracle</a> : A basic tutorial to setting up Ruby and Ruby's Oracle bindings on a Hewlett Packard UNIX based machine.</p>
<p><a href="http://wiki.rubyonrails.org/rails/pages/Oracle">Rails and Oracle from the official Rails wiki </a>: A whole ton of (older) information about Rails and Oracle integration.</p>
<p><a href="http://blogs.missiondata.com/?p=26">Ruby Oracle DBI ActiveRecord in 7 steps</a> : An up to date tutorial on how to get ActiveRecord running with an Oracle database in 7 easy(ish) steps.</p>
