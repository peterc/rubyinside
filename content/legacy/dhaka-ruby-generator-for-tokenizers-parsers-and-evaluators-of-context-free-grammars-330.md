<p><a href="http://dhaka.rubyforge.org/">Dhaka</a> is a set of tools written in Ruby by Mushfeq Khan that can generate tokenizers, parsers, and evaluators of context-free grammars (<a href="recursive-descent-parser-for-ruby-300.html">my own shabby contribution</a> in this field was posted to RubyInside recently!). Dhaka's ultimate goal is to make a pure Ruby, flexible parser generator. From the <a href="http://dhaka.rubyforge.org/">official site</a>:</p>
<blockquote><p>Dhaka provides DSLs specialized for specifying grammars, tokenizers and evaluation rules. It can take a grammar and generate a finite-state automaton for parsing it. The generated parser can then be 'compiled' to ruby source-code so that it need not be generated again. Dhaka also supports exporting generated parsers to the GraphViz dot format for visualizing states and actions.</p></blockquote>
<p>A couple of <a href="http://dhaka.rubyforge.org/example1.html">interesting</a> <a href="http://dhaka.rubyforge.org/example2.html">examples</a> of how Dhaka can be used to give simple solutions to <a href="http://rubyquiz.com/">Ruby Quiz</a> problems are also given.</p>
