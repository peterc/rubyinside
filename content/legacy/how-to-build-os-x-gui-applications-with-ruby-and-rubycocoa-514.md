<p><img src="/assets/2007/06/osxguirubycocoa.jpg" height="161" width="322" border="1" hspace="4" vspace="4" alt="Osxguirubycocoa"></p>
<p>Erik Kastner has put together a great tutorial showing you <a href="http://metaatem.net/2007/05/27/your-first-few-days-on-rubycocoa">how to build a graphical OS X app using Ruby and RubyCocoa</a>. For a less wordy approach, there's also <a href="http://images.metaatem.net/hello_world_newcocoa2.mov">a short screencast</a> which shows you the whole process from start to finish.</p>
<p>With the continual improvements of the <a href="http://rubycocoa.sourceforge.net/HomePage">RubyCocoa</a> library, and Apple's <a href="http://developer.apple.com/leopard/overview/apptech.html">commitment</a> to Ruby for Cocoa development in OS X 10.5, Ruby is rapidly becoming a viable alternative to Objective C and Java for developing OS X apps.<br>
<img src="/assets/2007/06/rubycocoacom.jpg" height="280" width="317" border="1" hspace="4" vspace="4" alt="Rubycocoacom"></p>
<p>Another great resource is <a href="http://www.rubycocoa.com/">RubyCocoa.com</a>, a site that features several RubyCocoa tutorials and articles.</p>
