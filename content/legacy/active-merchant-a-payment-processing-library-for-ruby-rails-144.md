<p><a href="http://home.leetsoft.com/am/">Active Merchant</a> is a payment processing library for Rails developed by the geniuses behind Rails powered e-commerce system, <a href="http://www.shopify.com/">Shopify</a>. It's under active development with support for different payment processor gateways being added regularly. So far it supports:</p>
<ul>
<li>Moneris</li>
<li>Authorize.net</li>
<li>TrustCommerce</li>
<li>LinkPoint</li>
<li>Psigate</li>
</ul>
<p>They want to integrate with WorldPay and ChronoPay next. The syntax for using Active Merchant and its descendent classes is ridiculously easy, so <a href="http://home.leetsoft.com/am/">check it out</a> (<em>code examples viewable there</em>).</p>
