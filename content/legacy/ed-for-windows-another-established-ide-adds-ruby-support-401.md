<span style="font-size:0pt;"><br>
</span>I'm not a big IDE or a Windows user myself, so getting me to review a Windows-based IDE could be quite tough. However, the creator of ED, Neville Franks, is an Australian-based independent software developer (trading as <a href="http://www.getsoft.com/">Soft As It Gets</a>) and wrote such a nice e-mail that I felt obliged to take a look.<br>
ED is a Windows-only editor with over 20 years' of history, having first been commercial released in the 80s, crammed with features a lot of developers seem to love, and with support for about twenty different programming languages out of the box. The latest is Ruby which Neville has so far been impressed with. ED performs code completion (ending of blocks, if statements, etc), syntax highlighting, class navigation, and all of the features you'd usually expect an IDE to boast. Neville has written <a href="http://blog.surfulater.com/2007/02/21/write-ruby-code-faster-with-ed-for-windows/">a comprehensive blog entry covering ED's support for Ruby</a>, where he demonstrates each of these features.</p>
<p>ED is a commercial product but there's a fully-featured <a href="http://www.getsoft.com/download_trial_version.html">30-day trial version available</a> to download. If you're still looking for a Ruby IDE on Windows that suits your tastes, give it a try.</p>
