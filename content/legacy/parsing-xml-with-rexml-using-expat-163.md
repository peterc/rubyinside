<p><a href="http://expat.sourceforge.net/">Expat</a> is the recognized big daddy of XML parsing. It's a stream-based XML parser written in C and, as a library, is used for XML parsing functions by many languages. Rubyists have tended towards <a href="http://www.germane-software.com/software/rexml/">REXML</a>, however, a more flexible (though infinitely slower) parser. Sam Ruby, however, has <a href="http://www.intertwingly.net/blog/2006/07/30/REXML-on-Expat">come up with some techniques to get Ruby's REXML working with Expat</a>.</p>
