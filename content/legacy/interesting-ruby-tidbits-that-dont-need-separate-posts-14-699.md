<p><strong>Rubular - Online Ruby Regular Expression Editor</strong></p>
<p><img src="/assets/2008/01/rubular.png" width="316" height="237" alt="rubular.png"></p>
<p><a href="http://www.rubular.com/">Rubular</a> is a Web-based regular expression editor and tester for Ruby developers, and particularly well suited for those who don't find regular expressions particularly easy. Very slick.</p>
<p><strong>FXRuby Book Available</strong></p>
<p><img src="/assets/2008/01/fxruby.png" width="157" height="188" alt="fxruby.png"></p>
<p>Harmon Vinson reports that a new book is available in beta form from the Pragmatic Programmers, namely "<a href="http://www.pragprog.com/titles/fxruby">FXRuby: Create Lean and Mean GUIs with Ruby.</a>" As you can expect, the book covers how to create GUI applications using Ruby and the cross-platform <a href="http://www.fox-toolkit.org/">FOX widgets library</a> using FXRuby. The only downside is that FOX, so far, has no native support for OS X's default Aqua window manager, but will work through X11.</p>
<p><strong>Ruport Book Available</strong></p>
<p><img src="/assets/2008/01/320_1700117.jpg" width="183" height="240" alt="320_1700117.jpg"></p>
<p><span style="font-style: italic;">The Ruport Book</span>, "Your Guide to mastering Ruby Reports", is <a href="http://www.lulu.com/content/1700117">now available to buy</a> in both PDF and print form. The book covers the use of the popular <a href="http://rubyreports.org/">Ruport Ruby Reports</a> library and is by Gregory Brown and Michael Milner. If you're already using the Ruport library, this is a must buy.</p>
<p><strong>Encrypted Cookies for Rails 2</strong></p>
<p>Aaron Bedra takes a look at <a href="http://aaronbedra.com/2008/1/13/baking-the-special-kind-of-cookies">how you can get Rails to encrypt the cookies</a> that go between your application and its users.</p>
<p><strong>Faster Rails Routing (6.4x Faster!) On The Way</strong></p>
<p>Michael Klishin writes in with news that Oleg Andreev has come up with <a href="http://novemberain.com/2008/1/17/routes-recognition">a smart way to radically increase the performance of Rails' routing code</a>.</p>
<p><strong>Free Git Hosting</strong></p>
<p>Johan Sørensen wanted everyone to know about his new open-source <a href="http://gitorious.org/">Gitorious</a> project, which provides free Git repository hosting. As many Rails coders have switched from Subversion to Git recently, this deserves a mention on Ruby Inside. It is also worth giving <a href="http://github.com/">Github</a>, a similar site by Tom Preston-Werner and Chris Wanstrath, a mention although it is not accepting signups just yet.</p>
