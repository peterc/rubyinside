<p><strong>Serving PDFs with Rails using Inkscape</strong><strong> </strong>Satya X has written "<a href="http://www.thesatya.com/blog/2007/09/inkscaperailspdf.html">Serving PDFs with Rails using Inkscape</a>," an article that goes into amazing detail about how to set up a system where customized PDFs can be created and served from a Rails application (or any Ruby app, with amendments). Ever wanted to set up your own Pragmatic Programmers'-esque PDF stamping system for selling e-books? Now you can do it for free.<strong> </strong><strong>Ruby on Rails vs DJ Ango</strong>The <a href="http://www.railsenvy.com/">RailsEnvy</a> guys are at it again with another Rails vs X "commercial." This time it's <a href="http://www.railsenvy.com/2007/9/10/ruby-on-rails-vs-django-commercial-7">Ruby on Rails vs Django</a>.. with stellar hip hop beats.<object height="350" width="425"><param value="http://www.youtube.com/v/PLUS00QrYWw" name="movie">
<param value="transparent" name="wmode">
<embed src="http://www.youtube.com/v/PLUS00QrYWw" height="350" width="425" wmode="transparent" type="application/x-shockwave-flash"></embed></object><strong>Ruby On The iPhone?</strong>Giles Bowkett writes with news that "Ruby runs on the iPhone" and presents these <a href="http://www.modmyiphone.com/nativeapps/the-list/">two</a> <a href="http://iphone.natetrue.com/">sites</a> as evidence (see ruby-1.8.6.tar.gz on the second). Giles has no iPhone, and nor do I, so we can't test it, but for those of you with such a device, it might be do-able. Please add any reports regarding iPhone Ruby operability as comments here.<strong>SOAP4R</strong>Never gotten familiar with Ruby's abilities with SOAP? Mark Thomas has put together a really tight <a href="http://markthomas.org/2007/09/12/getting-started-with-soap4r/">beginner's tutorial to the SOAP4R library</a>.<strong>Rubinius Sprint</strong>Last week, the team behind <a href="http://rubini.us/">Rubinius</a> - one of many promising attempts at creating a new Ruby implementation - had a <a href="http://blog.brightredglow.com/2007/9/17/rubinius-sprint-retrospective">major development sprint</a>. Lots of things got done, and Charles Nutter (of JRuby fame) even dropped in to help. Along with the rest of the Ruby community, we want to cheer on Rubinius and look forward to reporting on its beta releases in the near-future (hopefully!).</p>
