<p><img src="/assets/2008/12/radvent2008.png" width="89" height="85" alt="radvent2008.png" style="float:left; margin-right:12px; margin-bottom:12px;"> If you read Ruby Inside in 2006, you might remember <a href="advent2006/index.html">our 2006 Ruby Advent Calendar</a>. It featured the <a href="advent2006/1-amazon-s3.html">first look</a> at the then-new AWS::S3 library, offered a <a href="advent2006/7-coloring.html">Ruby code formatter</a> for blog posts (which I still use here!), some <a href="advent2006/24-wallpapers.html">Ruby wallpapers</a>, and a lot more.</p>
<p>2007 had no Ruby advent calendar, unfortunately, but this year Lakshan Perera, a Sri Lankan Ruby developer, has put together <a href="http://advent2008.hackruby.com/">a good-looking 2008 Ruby Advent Calendar</a>. Day one looked at <a href="http://advent2008.hackruby.com/past/2008/12/1/day_1_simple_blog_engine_for_ruby/">rolling out a Sinatra and Sequel powered blog</a>. Day two covered <a href="http://advent2008.hackruby.com/past/2008/12/2/deploying_ruby_apps_with_passenger_/">deploying Ruby apps with Phusion Passenger</a> (which <a href="http://www.railsinside.com/news/169-passenger-204-released-now-with-global-queuing-and-background-process-support.html">just hit version 2.0.4</a>). And day three <a href="http://advent2008.hackruby.com/past/2008/12/3/hoptoad_for_better_exception_notification/">looks at exception notification service, Hoptoad</a>.</p>
<p>The advent calendar has a feed at <a href="http://advent2008.hackruby.com/feed"><em>http://advent2008.hackruby.com/feed</em></a> if you want to follow it right up until Christmas Day - you can also follow <a href="http://twitter.com/rubyadvent">@rubyadvent</a> on Twitter if that's more your scene.</p>
