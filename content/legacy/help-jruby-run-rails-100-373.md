<p>Charles Nutter is <a href="http://headius.blogspot.com/2007/01/end-is-nigh-help-squash-rails-bugs.html">asking for help in squashing Rails bugs with JRuby</a>. It seems full support for Ruby on Rails with JRuby is only just over the horizon, with over 90% coverage of ActiveRecord and ActiveSupport so far. Charles provides instructions on how to set up Rails with JRuby and run your own tests.</p>
<p>Once JRuby supports Ruby on Rails 100%, a second deployment option for thousands of Rails applications suddenly becomes available, and with the <a href="http://headius.blogspot.com/2007/01/jruby-compiler-in-trunk-and-ready-to.html">impressive performance gains and optimizations becoming available in JRuby</a>, significantly faster Web apps could soon result.</p>
