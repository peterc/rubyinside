<p>It's only the first release candidate, but the <a href="http://weblog.rubyonrails.com/2006/11/23/rails-1-2-release-candidate-1">Rails 1.2 gravytrain is now pulling into the station</a>. The biggest new features include the full suite of REST goodies, a freshly-written Routing system, and significantly improved Unicode support.</p>
<p><em>An aside: I am wondering whether news such as this, which was already posted to the popular official Rails blog, should even be posted to Ruby Inside. I assume all Ruby Inside readers, if interested in Rails, would already be reading some of the important Rails blogs out there.. or are there people who only read Ruby Inside for their Ruby / Rails news? Please post a comment if you're not subscribed to the official Rails blog and would still like to see bulletins like this here.</em></p>
