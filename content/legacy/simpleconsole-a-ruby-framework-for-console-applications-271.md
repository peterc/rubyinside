<p>I think t0fuu has hit upon gold with <a href="http://t0fuu.wordpress.com/2006/09/16/building-console-apps-with-simpleconsole/">SimpleConsole</a>, his new Ruby framework for console applications. Imagine Ruby on Rails, but entirely for command line scripts, and that's SimpleConsole. It has automatic options parsing, before and after filter support, has logic and view separation, and a routing system. It's real thinking out of the box, and with more work it could be an incredibly powerful and productive way to knock up quick scripts for the command line.</p>
