<p>I know.. PHP. Don't stone me just yet! Eric Rollins presents <a href="http://eric_rollins.home.mindspring.com/pgen/">some interesting work relating to generating PHP code with Ruby</a>.</p>
<p>It's very much opposite to the whole way Rails works. You supply the system with XML files describing your schema and database setup, and the Ruby scripts pump out PHP files to produce a 'scaffolded' type application that can work with the data. The results are very similar to basic Rails scaffolding, but in PHP. The source code for the code generator is available for free download.</p>
