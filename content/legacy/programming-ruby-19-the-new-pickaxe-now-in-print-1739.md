<p><img src="/assets/2009/04/pickaxe2000.png" width="137" height="166" alt="pickaxe2000.png" style="float:left; margin-right:12px; margin-bottom:12px; border:1px #000000 solid;">Whether you love it or not, as a Rubyist you probably have a copy of <em>Programming Ruby</em> (also known as <em>The Pickaxe</em>) floating about. It was the first English language Ruby book to be published and was instrumental in boosting Ruby's popularity in the early <em>noughties</em>.</p>
<p>Now, after quite some time, <a href="http://www.pragprog.com/titles/ruby3/programming-ruby-1-9">the latest, third, edition of <em>Programming Ruby</em></a> has gone into print and is now available in both print and PDF/e-book formats. Direct from the publisher, The Pragmatic Programmers , you can pick it up as a eBook + Paper Book package for $59.95, an eBook only for $25, or the print version for $49.95. If you want to save some pennies, Amazon <a href="http://www.amazon.com/Programming-Ruby-1-9-Pragmatic-Programmers/dp/1934356085/">has the print copy for $32.97</a> (not an affiliate link).</p>
<p>The third edition of the Pickaxe is notable as the only complete reference for Ruby 1.9 (and the standard library) so if you want to be up to speed, fill your boots. O'Reilly's <a href="http://oreilly.com/catalog/9780596516178/toc.html"><em>The Ruby Programming Language</em></a>, by David Flanagan and Yukihiro "Matz" Matsumoto also covers 1.9 but isn't quite as up to date and presents Ruby in a more walkthrough fashion. <em>Programming Ruby</em> now clocks in at a stomping 960 pages, quite a growth on the 800 page second edition..!</p>
