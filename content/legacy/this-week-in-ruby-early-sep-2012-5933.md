<p>Welcome to this week's Web-based syndication of <a href="http://rubyweekly.com/">Ruby Weekly</a>, the Ruby e-mail newsletter (Now at 16,300 subscribers! C'mon.. check it out ;-)).</p>
<p>Highlights include: an announcement over the dates for this year's Rails Rumble, releases of Active Admin 0.5 and Bundler 1.2, as well as Aaron Patterson looking at a difference in Object#respond_to? in the forthcoming Ruby 2.0.</p>
<h3>Featured</h3>
<div style="margin: 16px 0px;" class="item">
<p><a href="http://blog.railsrumble.com/2012/09/13/registration-dates-competition-details" style="font-weight: bold;">The Rails Rumble is Back! Dates and Details Announced</a><br>
The popular 48 hour Ruby development contest is back and will run this October 13th and 14th. Registration opens on October 1st. Learn all about it here.
</p>
</div>
<div style="margin: 16px 0px;" class="item">
<p><a href="http://reverbhq.com/blog/2012/08/activeadmin-0-5-0-is-released/" style="font-weight: bold;">Active Admin 0.5.0 Released</a><br>
Active Admin is a popular administrative interface generation system for Rails and a new version has been a long time coming. 0.5 introduces batch actions (i.e. select multiple rows and delete them all), a customizable root route, and more view components and factories.
</p>
</div>
<div style="margin: 16px 0px;" class="item">
<p><a href="http://gembundler.com/v1.2/whats_new.html" style="font-weight: bold;">Bundler 1.2 Released</a><br>
The vital dependency management tool gets an update and now supports specifying Ruby versions in Gemfiles, local git repos, and using 'bundle package' to package git and local path dependencies.
</p>
</div>
<div style="margin: 16px 0px;" class="item">
<p><a href="http://www.akitaonrails.com/2012/09/03/why-documentary-at-rubyconf-2012-denver" style="font-weight: bold;">'Why The Lucky Stiff' Documentary to Screen at RubyConf 2012</a><br>
Fabio Akita talks about Why The Lucky Stiff, a creative and enigmatic Rubyist who disappeared from the Web back in 2009, and reveals that a documentary about his work is under production and due to be shown at RubyConf 2012.
</p>
</div>
<div style="margin: 16px 0px;" class="item">
<p><a href="http://www.ruby-lang.org/en/news/2012/09/07/confoo-2013call-for-papers-is-now-open/" style="font-weight: bold;">ConFoo 2013 Call for Papers is Open</a><br>
The Call For Papers is only open for 10 more days and the conference itself is in February 2013.
</p>
</div>
