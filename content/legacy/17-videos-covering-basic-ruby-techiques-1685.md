<p><img src="/assets/2009/04/vidgfx.gif" width="100" height="110" alt="vidgfx.gif" style="float:left; margin-right:12px; margin-bottom:12px;"><a href="http://www.tekniqal.com/">Tekniqal.com</a> is a site offering a series of 17 Basic Ruby tutorials in screencast form. In a way, it's like <a href="http://railscasts.com/">Railscasts</a>, but focusing solely on basic Ruby techniques. So far there are 17 tutorials covering basic topics from whitespace and identifiers through to symbols and hashes, but it looks like there'll be more in future.</p>
<p>These videos probably won't be of much direct use to the average Ruby Inside reader directly, but if you have developers under you or friends who need to learn Ruby and you want them to get a quick fire, bite by bite introduction to some of the concepts involved, point them to Tekniqal.</p>
<p>The videos so far are:</p>
<p style="margin-left: 1.5em">1. <a href="http://www.tekniqal.com/2009/2/Identifiers-in-Ruby">Identifiers</a><br>
2. <a href="http://www.tekniqal.com/2009/2/Whitespace-In-Ruby">Whitespace</a><br>
3. <a href="http://www.tekniqal.com/2009/3/Duck-Typing-in-Ruby">Dynamic Typing</a><br>
4. <a href="http://www.tekniqal.com/2009/3/Variable-Scope-in-Ruby">Variables - Scope</a><br>
5. <a href="http://www.tekniqal.com/2009/3/Methods-in-Ruby">Methods</a><br>
6. <a href="http://www.tekniqal.com/2009/3/Conditions-in-Ruby">Conditions</a><br>
7. <a href="http://www.tekniqal.com/2009/3/Loops-in-Ruby">Loops</a><br>
8. <a href="http://www.tekniqal.com/2009/3/String-Delimiters-in-Ruby">String Delimiters</a><br>
9. <a href="http://www.tekniqal.com/2009/3/Symbols-in-Ruby">Symbols</a><br>
10. <a href="http://www.tekniqal.com/2009/3/Strings-and-Mutability-in-Ruby">Strings and Mutability</a><br>
11. <a href="http://www.tekniqal.com/2009/4/Ranges-in-Ruby">Ranges</a><br>
12. <a href="http://www.tekniqal.com/2009/4/Creating-Arrays-in-Ruby">Creating Arrays</a><br>
13. <a href="http://www.tekniqal.com/2009/4/Accessing-Arrays-in-Ruby">Accessing Arrays</a><br>
14. <a href="http://www.tekniqal.com/2009/4/Manipulating-Arrays-in-Ruby-Part-1-of-2">Manipulating Arrays (Part 1)</a><br>
15. <a href="http://www.tekniqal.com/2009/4/Manipulating-Arrays-in-Ruby-Part-2-of-2">Manipulating Arrays (Part 2)</a><br>
