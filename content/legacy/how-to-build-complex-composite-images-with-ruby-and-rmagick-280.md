<p>Corban Brook presents a <a href="http://schf.uc.org/articles/2006/10/18/render-greatlooking-collages-with-ruby-and-rmagick">step-by-step guide to producing 'great looking collages'</a> using Ruby and the RMagick graphics library. Rather than be stuck with a dry picture-by-picture view, Brook uses RMagick to great effect to produce a complex, dynamic image that looks as if it were put together by hand. If for nothing else, this is a great look at some reasonably advanced RMagick code and a great way to learn about a few of the more arcane methods it offers.</p>
