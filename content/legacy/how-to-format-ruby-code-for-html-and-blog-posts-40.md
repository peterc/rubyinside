<p>You might have noticed this blog has nice, syntax colored code excerpts, as does <a href="http://www.bigbold.com/snippets/">Code Snippets</a>. Jim Morris looks at <a href="http://blog.wolfman.com/articles/2006/05/26/howto-format-ruby-code-for-blogs">how you can pull this off</a> in a few different ways. One quick and easy way is to use <a href="http://syntax.carldr.com/">syntax.carldr.com</a>, a Web site that converts any Ruby code you paste into the correct syntax colored HTML. Currently I use TextMate and an external script using the <em>syntax</em> gem, but might use that site instead as it's easier!</p>
