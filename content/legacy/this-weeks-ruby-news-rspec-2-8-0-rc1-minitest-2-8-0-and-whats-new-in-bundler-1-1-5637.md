<p>After <a href="this-weeks-ruby-news-ruby-1-9-3p0-omniauth-1-0-maglev-1-0-and-more-5580.html">last week's bumper set of releases</a> the Ruby world seems a lot quieter this week :-) (Even here. I've been hidden away teaching my <a href="http://rubyreloaded.com/">Ruby Reloaded course!</a>)</p>
<h3>Headlines</h3>
<div style="margin: 16px 0px;" class="item">
<p><a href="http://www.xydo.com/toolbar/37869317?subject_id=503" style="font-weight: bold;">RSpec 2.8.0 RC1 Released</a><br>
The next significant release of RSpec is afoot and its first release candidate is now out. Key improvements come to configuration (and being able to override it from the command line) and running examples in random (and pseudo-random) order.
</p>
</div>
<div style="margin: 16px 0px;" class="item">
<p><a href="http://rubyinstaller.org/downloads" style="font-weight: bold;">Ruby 1.9.3-p0 RubyInstaller for Windows Available</a><br>
RubyInstaller is a popular route to installing Ruby for Windows users, and a new version based on the all-new Ruby 1.9.3-p0 is now out.
</p>
</div>
<div style="margin: 16px 0px;" class="item">
<p><a href="http://newrelic.com/server_monitoring" style="">Ruby Webapp Performance Management Service New Relic Adds Free Server Monitoring</a></p>
</div>

<h3>Articles and Tutorials</h3>
<div style="margin: 16px 0px;" class="item">
<p><a href="http://patshaughnessy.net/2011/11/5/besides-being-faster-what-else-is-new-in-bundler-1-1" style="font-weight: bold;">Besides Being Faster, What Else is New in Bundler 1.1?</a><br>
Recently, Pat Shaughnessy did a great post explaining why the forthcoming Bundler 1.1 will be faster than 1.0. This time around, he looks at some new features, including bundle outdated, bundle clean, specifying install paths, and creating bundles that will work on machines that don't have Bundler installed.
</p>
</div>
<div style="margin: 16px 0px;" class="item">
<p><a href="http://weblog.therealadam.com/2011/11/06/the-current-and-future-ruby-platform/" style="font-weight: bold;">The Current and Future Ruby Platform</a><br>
Adam Keys shares his thoughts on the long-term future potential of Ruby as a platform and where it's likely and unlikely to go.
</p>
</div>
<div style="margin: 16px 0px;" class="item">
<p><a href="https://github.com/devalot/ror-example" style="font-weight: bold;">A Self Guided Rails 3.1 Tutorial in an App</a><br>
An interesting app built by Peter Jones that aims to be a self guided Rails 3.1 tutorial. The README is extensive and explains how to get started. There are also some reader exercises designed to stretch your coding muscles.
</p>
</div>

<h3>Libraries and code</h3>
<div style="margin: 16px 0px;" class="item">
<p><a href="http://engineering.gowalla.com/2011/11/03/likeable/" style="font-weight: bold;">Likeable: Storing 'Liked' Objects with Redis</a><br>
Likeable is a new open-source Ruby library built by Gowalla to store 'likes' on items. It works with ActiveRecord objects but can be used with any Ruby object that implements an #id method. If you have Redis ready and waiting, it looks pretty easy to get going.
</p>
</div>
