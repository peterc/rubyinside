<p>Exactly one year ago <a href="http://www.loudthinking.com/">David Heinemeier Hansson</a> and the rest of the Rails core team (smaller back then!) <a href="http://weblog.rubyonrails.com/2005/12/13/rails-1-0-party-like-its-one-oh-oh/">unveiled Rails 1.0</a>, the first 'production quality' release of Ruby on Rails. Alongside the release came an entirely new <a href="http://www.rubyonrails.com/">Rails Web site</a>, the same as the one we're familiar with now. The next day, December 14, DHH followed up with <a href="http://www.loudthinking.com/arc/000550.html">some brief insight</a> and wondered: "<em>I can't wait to see where this will all go in 2006.</em>"</p>
