<p><img style="margin: 0px 12px 12px 0px; float: left" src="/assets/2008/08/github.png" alt="" width="172" height="76"><a href="http://www.github.com">Github</a> is a great resource for finding new projects within the Ruby community. It has become an extremely popular place for Ruby and Rails developers to congregate lately, so I wanted to list some of the new projects, and some of the updated ones, that I have found interesting and that are too small for their own blog post. Let us know if you like this as we might turn it into a regular series on Ruby Inside!</p>
<p>This month's picks:</p>
<ul>
<li>
<strong><a href="http://github.com/tyler/uppercut/">Uppercut</a></strong> - A simple DSL for creating Jabber agents within Ruby. Seems quite young but full of potential.</li>
<li>
<a href="http://github.com/nakajima/aintablog/tree/master"><strong>AintABlog</strong></a> - Open-source tumblog application. Imagine your own <a href="http://tumblr.com">tumblr</a> account, but really customizable.</li>
<li>
<a href="http://github.com/technicalpickles/factory_girl_on_rails"><strong>factory_girl_on_rails</strong></a> - If you use <a href="http://github.com/thoughtbot/factory_girl/" target="_blank">factory_girl</a> <em>(like I do)</em> then this will provide you with automatic inclusion of test/factories/* and also new generators for creating factories.</li>
<li>
<a href="http://github.com/flogic/object_daddy" target="_blank"><strong>object_daddy</strong></a> - A nice alternative to factory_girl. It helps you kill rails fixtures, keep your code <abbr title="Don't Repeat Yourself">DRY</abbr> and reduce the complexity of your tests.</li>
<li>
<a href="http://gist.github.com/6443"><strong>Chris Wanstrath's Ruby Hoedown Keynote</strong></a> - An unusual entry for Github, but as Chris runs GitHub, this makes sense. It's a transcript of Chris's great keynote presentation at the recent Ruby Hoedown conference.</li>
<li>
<a href="http://github.com/lazyatom/backchat/tree/master"><strong>Backchat</strong></a> - a one-file Merb application that allows you to quickly add commenting (think <a href="http://www.disqus.com">disqus</a>) functionality on to any site.</li>
<li>
<a href="http://github.com/metaskills/homemarks/tree/master"><strong>HomeMarks</strong></a> - a Rails application that provides "start page" type functionality. A great example of Rails code using unobtrusive JavaScript. Very slick.</li>
<li>
<a href="http://github.com/blog/143-inline-file-editing"><strong>Inline File Editing</strong></a> - Not a project, but a new Github feature. You can now edit files in your repositories within the browser and Github deals with the housekeeping. Perfect for the most minor tweaks.</li>
<li>
<a href="http://github.com/jicksta/adhearsion/tree/master"><strong>Adhearsion</strong></a> - A popular VoIP framework (a bit like a "Rails for telephony") that has just migrated over to using GitHub.</li>
</ul>
