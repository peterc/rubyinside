<p><a href="http://shanesbrain.net/">Shane Vitarana</a> has launched <a href="http://shanesbrain.net/articles/2006/07/18/rails-stats-plugin-or-show-me-those-pretty-graphs">a new Rails plugin called Rails Stats</a>. It's a graphical version of the <em>rake stats</em> statistics. It generates graphs showing information about your application, and is powered by Geoffrey Grosenbach's great <a href="http://nubyonrails.com/pages/gruff">Gruff Graphs</a> plugin.<br>
Install like so:</p>
<pre><code>script/plugin install http://shanesbrain.net/svn/rails/plugins/rails_stats</code>
</pre>
<p>And then load <code>/rails_stats</code> on your running Rails application. <a href="http://shanesbrain.net/articles/2006/07/18/rails-stats-plugin-or-show-me-those-pretty-graphs">More info here.</a></p>
