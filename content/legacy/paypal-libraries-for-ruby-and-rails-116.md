<p>I initially thought there was only one PayPal library for Ruby, but apparently not. Here's all those I've found:</p>
<p><a href="http://dist.leetsoft.com/api/paypal/">Paypal</a> by Tobias Luetke - Primarily developed to help integrate PayPal with Rails applications. It works with PayPal's IPNs (Instant Payment Notifications). Source code demonstrations included.</p>
<p><a href="http://rubyforge.org/projects/vpaypal">VPayPal</a> - A new library (released February 2006) that supports both Express Checkout API and the direct checkout API.</p>
<p><a href="http://www.elctech.com/products_ruby_paypal.shtml">RoR::PayPal</a> - A library developed by ELC Technologies and made available under the LGPL. It doesn't seem to support the Express Checkout API but presents another option and has a detailed README.</p>
<p><a href="http://www.pranavbihari.com/articles/2005/12/02/testing-paypal-web-services-with-ruby-soap4r">Testing PayPal Web Services with Ruby soap4r</a> - This isn't a library but Pranav Bihari looks at the internals of talking between Ruby and PayPal. This could be useful for developing your own library.</p>
