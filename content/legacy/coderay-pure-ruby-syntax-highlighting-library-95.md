<p><a href="http://coderay.rubychan.de/">CodeRay</a> is a totally new syntax highlighting library written in Ruby. It deals with Ruby, C, Delphi, HTML, RHTML, and Nitro-XHTML out of the box, with plans to support YAML, SQL, Python, Perl, PHP, and Java. It's quite customizable and easy to use, as seen in the code in the image above. Nice work!</p>
