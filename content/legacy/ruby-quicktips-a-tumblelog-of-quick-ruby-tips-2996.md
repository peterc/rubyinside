<p><img src="/assets/2010/02/ruby-quicktips.png" width="86" height="77" alt="ruby-quicktips.png" style="float:left; margin-right:12px; margin-bottom:12px;"><a href="http://rubyquicktips.tumblr.com/">Ruby Quicktips</a> is a Tumblr-powered tumblelog (think of a blog but in bite-sized chunks) by Daniel Pietzsch that presents a growing array of Ruby related tips and interesting code snippets. Daniel seems keen for you to contribute, and you can do so on <a href="http://rubyquicktips.tumblr.com/submit">its submissions</a> page. There's an <a href="http://rubyquicktips.tumblr.com/post/380495736/about-ruby-quicktips">"about us" post</a> if you want to learn more in general.</p>
<p>Daniel has done a good job of curating the 20 or so tips that he's put up over the last couple of months, and I bet with your help, it could become a significant resource. Picking up a few tips each week is a great way to keep your mind fresh and the ideas flowing! The tips posted so far are mostly quite basic, but with more submissions, a wider gamut of skill levels could be covered.</p>
<p>A selection of the tips posted so far:</p>
<ul>
<li><a href="http://rubyquicktips.tumblr.com/post/342527837/console-tip-retrieve-the-last-return-value-with">Retrieve the last return value with underscore (_)</a></li>
<li><a href="http://rubyquicktips.tumblr.com/post/382482752/give-it-a-try">Using the "try" method</a></li>
<li><a href="http://rubyquicktips.tumblr.com/post/354231466/date-manipulation">Date manipulation</a></li>
<li>
<a href="http://rubyquicktips.tumblr.com/post/334692052/disabling-input-fields-on-a-form-based-on-a-condition">Disabling input fields on a form based on a condition</a> (Rails)</li>
<li><a href="http://rubyquicktips.tumblr.com/post/277330553/all-and-any">all? and any?</a></li>
</ul>
<p>Ruby Quicktips also has <a href="http://twitter.com/rubyquicktips">a Twitter account</a> you can follow to be notified when the latest tip has been published. There's also <a href="http://rubyquicktips.tumblr.com/rss">an RSS feed</a> if you prefer that.</p>
