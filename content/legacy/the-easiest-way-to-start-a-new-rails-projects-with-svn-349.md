<p>You're about to start a new Rails project. Do you type <strong><em>rails &lt;project name&gt;</em></strong>? No, "<a href="http://blog.hasmanythrough.com/2006/12/28/stop-using-the-rails-command/">Stop using the rails command</a>", says Josh Susser. If you create a dummy / default Rails project that's set up just the way you like it with plugins, your own extensions, and all those carefully defined svn:ignore flags, you can just create a new Rails project with a simple <strong><em>svn copy</em></strong>. So simple, but I've totally missed this idea, and it's a real timesaver.</p>
