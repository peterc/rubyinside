<p>Arcadian Visions have released <a href="http://www.arcadianvisions.com/downloads/index.html#RubySpotlight">a plugin for Mac OS X's Spotlight that lets you quickly search </a>your Ruby source code. Once indexed, you can search by module, class, method, general contents, or code comments. <em>(Found via </em><em><a href="http://digitalhominid.apesseekingknowledge.net/">Have GNU, Will Travel</a></em><em>)</em></p>
<p><strong>(Added: Tread with caution! It seems one user is having problems installing this on their Intel x86 powered Mac. See comments.)</strong></p>
