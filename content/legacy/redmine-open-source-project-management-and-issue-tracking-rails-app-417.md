<p><a href="http://redmine.rubyforge.org/">redMine</a> is an open source project management and issue tracking Web app, created by Jean Philippe Lang and released under the GPL. There's a <a href="http://demo.redmine.org/">live demo</a> of the system available. Features include multiple user support, SVN and diff browsers, LDAP support, support for several different database systems, and more.</p>
