<p>Rick Olson and Josh Goebel have teamed up to produce Beast, a lightweight but well styled forum. Rails has sorely been lacking a good, well styled, open source forum, but Beast is great. Check out <a href="http://beast.caboo.se/">the demonstration forum</a> or go <a href="http://svn.techno-weenie.net/projects/beast/">get the code for yourself</a>.</p>
