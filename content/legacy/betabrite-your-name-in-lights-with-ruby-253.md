<a href="http://tenderlovemaking.com/2006/09/28/new-ruby-betabrite-002/">BetaBrite</a> is a Ruby LCD sign library developed by Aaron Patterson. Since few people have LCD signs to play with, Aaron has set up a test-bed installation that you can use via DRb. Ingenious! Next I expect a Ruby <a href="http://www.blinkenlights.de/">blinkenlights</a> library and DRb hookup.<br>
