<p>You may have thought Ruby on Rails was enough to get Web applications developed quickly, but enter Hobo. <a href="http://hobocentral.net/blog/">Hobo</a> makes the process of creating Web applications and prototypes even quicker. For example, out of the box, with no lines of code written, you get a dummy app with a user signup, login, and authentication system.</p>
<p>Hobo can be installed as a gem so you can create new applications from scratch with the <em>hobo</em> command line tool, or as a plugin so you can add Hobo features to existing applications. Learn more in <a href="http://hobocentral.net/blog/getting-started/">the installation guide</a>. Hobo also includes a templating system and mark up language called DRYML (Don't Repeat Yourself Markup Language) that allows you to include things using custom defined HTML tags, rather than <em>&lt;%= %&gt;</em> blocks.<br>
There's quite a lot to Hobo, so you'll want to go through its comprehensive <a href="http://hobocentral.net/blog/">official site</a> and watch <a href="http://screencasts.hobocentral.net/hobo-pod-screencast.mov">the Hobo screencast</a> to get a real feel for it where a classified ads app is created within minutes.</p>
