<p>Are you a OS X dwelling, TextMate wielding, Rubyist? I am, but I'm next to useless with all of the advanced features (and by advanced I mean anything beyond Cmd+R). Lucky, then, that Derek Neighbors has started a super-duper series of TextMate Tip screencasts!</p>
<p>So far there are three videos, but I suspect he has more on the way, so if you like the look of them, subscribe to <a href="http://derekneighbors.com/feed/">his feed</a>:</p>
<ul>
<li>
<a href="http://derekneighbors.com/2009/03/textmate-tip-1-hashes/">TextMate Tip #1 (Hashes)</a> - Derek demonstrates how TextMate can make your life easier when dealing with Ruby hashes, primarily shortcuts to produce a hashrocket (=&gt;), key value pair shortcut, a new hash, etc.</li>
<li><a href="http://derekneighbors.com/2009/03/textmate-tip-episode-2-blocks/">TextMate Tip #2 (Blocks)</a></li>
<li><a href="http://derekneighbors.com/2009/03/textmate-tip-3-dont-leave-the-editor/">TextMate Tip #3 (Don't Leave The Editor)</a></li>
</ul>
<p>And as a bonus - because I love you guys <em>that</em> much - how about three older Ruby-related TextMate screencasts too?</p>
<ul>
<li>
<a href="http://macromates.com/screencast/ruby_quiz_screencast.mov"><strong>Ruby Unit Tests and More</strong></a> <strong>by James Edward Gray II.</strong> James walks through using TextMate's Ruby bundle to apply unit tests to code. It's from way back in 2006 so YMMV!</li>
<li>
<a href="http://drnicwilliams.com/2008/06/11/using-ruby-within-textmate-snippets-and-commands/"><strong>Using Ruby within TextMate snippets and commands</strong></a> <strong>by Dr Nic Williams.</strong> Dr Nic leads us on a 7 minute journey into using Ruby <em>within</em> TextMate snippets. This is real power user stuff.</li>
<li><strong><a href="https://peepcode.com/products/textmate-for-rails-2">TextMate for Rails 2</a> from PeepCode.</strong></li>
</ul>
