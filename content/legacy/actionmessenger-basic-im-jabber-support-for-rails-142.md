<p><a href="http://trypticon.org/software/actionmessenger/">Action Messenger</a>, by Trejkaz Xaoza, is a basic wrapper around Ruby's <a href="http://home.gna.org/xmpp4r/">xmpp4r</a> Jabber library that makes <a href="http://en.wikipedia.org/wiki/Jabber">Jabber</a> instant messaging services directly available to Rails applications. <em>(Jabber is an open, XML specification and framework for instant messaging. </em><em><a href="http://www.google.com/talk/">Google Talk</a></em><em> uses it, for example.)</em></p>
<p>It's still in its infancy and currently only supports basic message sending and receiving of messages, but with instant messaging rapidly merging into the plans of many Web applications, it's a good start. If this sort of functionality is on your wishlist, check it out.</p>
