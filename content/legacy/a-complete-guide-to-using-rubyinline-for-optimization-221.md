<p>Eric Hodel has put together <a href="http://segment7.net/projects/ruby/inline_optimization.html">a great guide to optimizing Ruby code with RubyInline and ruby-prof</a>, a code profiler.</p>
<blockquote>
<p><em>Here's how I optimized png.rb to make PNG saving over 100 times faster using RubyInline. This is a good model to follow to make any ruby program faster.</em></p>
<p><em>The basic steps are to optimizing code are:</em></p>
<p><em>   1. Profile<br>
2. Find the method taking the most time<br>
3. Try one change to make it faster<br>
4. Profile<br>
5. Check for improvement<br>
* No improvement? Not enough improvement? Remove your change, go to step 3<br>
* Improvement? Go to step 1 </em></p>
</blockquote>
