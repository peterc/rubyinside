Rodney Ramdas has put together <a href="http://www.pinupgeek.com/articles/2006/06/04/learn-ruby-app">a small application called "Learn Ruby"</a> that shows the freely available version of the Pickaxe Ruby book alongside an instance of irb, meaning you can work through the examples and tutorials more freely. Note that it's for OS X only.</p>
<p>If you want something similar, but which lets you use a Ruby interpreter <em>direct from your Web browser</em>, try out why's infamous and incredibly clever <a href="http://tryruby.hobix.com/">Try Ruby!</a></p>
